# YG-1 AI Catalog Test Matrix

기준: `yg1-ai-catalog` 내부 `__tests__` 와 `__test__` 계열 파일, `node_modules` 제외.

## 읽는 법

| 컬럼 | 의미 |
|---|---|
| 파일 | 테스트 파일 |
| 입력 텍스트 / 준비 상태 | 실제 문자열 입력 또는 mock/session/decision 준비 방식 |
| 단계별 로직 흐름 | 테스트가 어떤 순서로 상태를 만들고, 입력을 넣고, 결과를 판정하는지 |
| 기대 결과 | `expect(...)` 로 요구하는 결과 |
| 판단 기준 | pass/fail 이 갈리는 핵심 assertion 기준 |
