# 3D contouring / ball-nose finishing — Helical, CNCCookbook

Source: https://www.helicaltool.com/products/contour-finishers-4-flute-ball-tapered-neck
Source: https://www.cnccookbook.com/cnc-stepover/

- Ball-nose 4-flute variable-pitch tools for 3D contour finishing of molds/dies.
- Stepover chosen via scallop/cusp height; diminishing returns below ~1/8 of tool diameter.
- Typical finishing stepover: 5–12% D (cusp target 0.0002"–0.005").
- Short-neck tapered-neck tools maximize rigidity on deep 3D walls.
- Effective cutting diameter changes with Z engagement — compensate Vc.
