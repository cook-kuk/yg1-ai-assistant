# Machining HRSA / Inconel — Sandvik Coromant (excerpt)
Source: https://www.sandvik.coromant.com/en-gb/knowledge/materials/pages/hrsa.aspx

- Nickel-based superalloys (Inconel 718, 625) retain strength at high temperature and work-harden aggressively.
- Use Vc 20–40 m/min for solid carbide, 200+ m/min for SiAlON ceramics.
- Keep a positive chip load (fz 0.05–0.10 mm) so the edge cuts beneath the previous pass hardened layer.
- Trochoidal paths (ae 5–10% D, full ap) extend life dramatically.
- Rigid setup, sharp tools, and copious coolant (or high-pressure jet) are essential.
- Never dwell with the tool engaged — immediate work hardening and tool failure.
