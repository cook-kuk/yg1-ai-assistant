# VDI 3323 — Applicability of Hard Cutting Materials

Source: https://www.vdi.de/en/home/vdi-standards/details/vdi-3323-applicability-of-hard-cutting-materials-for-metal-cutting-processes
Also: https://www.machiningdoctor.com/charts/vdi-3323-material-groups/
Fetched: 2026-04-09

VDI 3323 is a German VDI guideline that complements ISO 513. It defines a finer-grained machinability classification of workpiece materials and a standardized way to describe the application suitability of carbide/CBN/PCD grades.

Key idea: the six ISO 513 main groups (P, M, K, N, S, H) are divided into numbered machinability classes (approx. 1..38). Each class groups materials that behave similarly under cutting and can therefore be served by the same grade/feed/speed recommendation.

Examples of VDI 3323 class ranges:
- Classes 1-7: non-alloy and low-alloy steels (ISO P)
- Classes 8-14: high-alloy / tool steels (ISO P)
- Classes 15-20: stainless steels (ISO M)
- Classes 21-25: cast iron (ISO K)
- Classes 26-28: non-ferrous / aluminum (ISO N)
- Classes 29-34: heat-resistant superalloys & titanium (ISO S)
- Classes 35-38: hardened steel (ISO H)

Most major tool makers map their carbide grade recommendations to VDI 3323 classes so that a user can look up "my material is VDI 3323 class 20 -> use grade X at feed/speed Y".
