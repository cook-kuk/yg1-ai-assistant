# ZrN — Eifeler, IVAC, Wikipedia
- Hardness ~2800 HV (PVD); smoother than TiN
- Better oxidation resistance than TiN at 500°C; stable to ~600°C
- Low COF, non-sticking — excellent for Al, Ti, non-ferrous, medical, food
- Light gold / pale brass color
- Thickness 1–4 µm
Source: https://www.eifeler.com/northamerica/en/pvd-coatings/zrn/
