# nACo (PLATIT nc-AlTiN/a-Si3N4) — PMC, ScienceDirect
- Nanocomposite: nc-(Al,Ti)N matrix + a-Si3N4 amorphous boundary
- Nanohardness 40–45 GPa (≈4100–4600 HV); E 450–550 GPa
- Max use temperature ~1200°C in air
- COF ~0.40 (fretting)
- Deposition: PVD (PLATIT π411PLUS LARC cathodic arc)
- Hardened steels, HRSA, die/mold
Source: https://pmc.ncbi.nlm.nih.gov/articles/PMC12073052/
