# Sandvik Coromant - CoroMill Plura (Solid Carbide End Mills)

Source: https://www.sandvik.coromant.com/en-us/tools/milling-tools/solid-carbide-end-mills/coromill-plura

## CoroMill Plura HD (Heavy Duty)
- First choice for heavy-duty applications in ISO P and ISO M up to 2xD roughing in steel/stainless
- Zertivo 2.0 PVD coating (AlCrN family)
- Internal coolant, optimized flute shape, stainless steel geometry with radial exit coolant
- Grades: P2BM (steel HD), M-geometry for stainless
- 4 flute, square / corner radius variants

## CoroMill Plura for Edging / Side-Milling / High-Feed Side Milling / Roughing with Chip Breaker
- Multiple sub-families targeting specific operations
- Diameter range typically 1-25 mm
- Available as cylindrical or Weldon shank

## CoroMill Plura Hard-Part Milling
- Up to 63 HRC hardened steels
- Ball nose and corner radius variants, TiAlN/AlCrN coatings

Applications: ISO P (steel), ISO M (stainless), some ISO H (hardened) - roughing, finishing, slotting, side milling, edging
