# CrN — Calico / BryCoat / Eifeler
- Hardness 1750–2500 HV
- Max operating temp ~700°C
- Friction vs steel 0.3–0.5; vs polymers ~0.2
- Thickness 1–10 µm (typ 2–4)
- Silvery metallic color
- Good for copper/brass, Ti, Al die-casting, forming tools (corrosion + low adhesion)
Source: https://www.calicocoatings.com/coating-data-sheets/chromium-nitride-pvd-crn-coating/
