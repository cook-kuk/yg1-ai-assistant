# AlTiN — Calico, BryCoat, TinCoat, Wikipedia
- Hardness 3300–3500 HV (nano-hardness ~38 GPa)
- Max working temperature 800–900°C in air (Al2O3 self-forming layer)
- Friction coefficient ~0.35–0.37 at RT
- Higher Al than TiAlN — better high-heat oxidation resistance
- Ideal for high-speed dry cutting, hardened steels, Ni alloys
