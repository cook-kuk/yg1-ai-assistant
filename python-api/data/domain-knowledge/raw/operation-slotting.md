# Slotting / full slot — Harvey Performance

Source: https://www.harveyperformance.com/in-the-loupe/successful-slotting-with-miniature-cutting-tools/
Source: https://www.harveyperformance.com/in-the-loupe/depth-of-cut/

- Slot: the entire cutter diameter (ae = 100% D) is engaged.
- Axial engagement (ap) must be conservative — start 25–50% D, reduce for small tools.
- Use 2–3 flute end mills for better chip evacuation.
- High-pressure/through-coolant strongly recommended; chip re-cutting = breakage.
- Feed reduction ~50% vs. side milling to compensate for two-flank engagement.
