# Plunge milling — Harvey Performance

Source: https://www.harveyperformance.com/in-the-loupe/plunge-and-pocket-milling-in-4140-steel-with-helicals-combination-feed-hem-mill/

- Plunge (Z-axis) milling drives cutting force into the spindle — great for long overhangs and chatter-prone setups.
- Requires center-cutting end mills or dedicated plunge/insert tools.
- Step-over between plunges: 75% D typical (leaves scallop cleaned by finishing pass).
- Feed per tooth (axial) typically 50% of a slotting fz starting point.
- Excellent alternative when helical ramp is impossible (very deep pockets, L/D > 4).
