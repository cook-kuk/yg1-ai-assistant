# Ramping / helical entry — Harvey Performance

Source: https://www.harveyperformance.com/in-the-loupe/ramping-success/
Source: https://www.harveyperformance.com/in-the-loupe/types-tool-entry/

- Ramping = simultaneous radial + axial motion; linear (angular) or circular (helical).
- Recommended ramp angle: 1–3° for most solid end mills; <1° for hard materials.
- Helical interpolation preferred for closed forms (pockets, bores).
- Helix diameter >110–120% of cutter diameter.
- Center-cutting end mills required; conventional 4-flute tools fine for ramping.
