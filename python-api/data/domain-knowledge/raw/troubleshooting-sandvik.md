# Sandvik Coromant — Milling Troubleshooting

Source: https://www.sandvik.coromant.com/en-us/knowledge/milling/troubleshooting-milling

## Vibration / Chatter
- Position cutter off-center to the left: thicker chip at entry, thin chip at exit → more constant cutting force direction.
- Cutter diameter should be 20–50% larger than width of cut.
- Reduce ap/ae, reduce overhang, use unequal-pitch cutters, reduce number of teeth in cut.

## Chip control / chip jamming / re-cutting of chips
- Exit from workpiece is the most sensitive phase; avoid thick chip at exit — drastically reduces carbide insert life.
- Use internal coolant / air blast to evacuate chips from pocket.
- Reduce number of effective flutes; increase chip pocket.

## Unsatisfactory surface finish
- Check runout, insert wear, feed per tooth, wiper insert usage.

## Burr formation
- In soft low-carbon steels BUE and burr are main issues.
- Use inserts with sharp edges (PVD coated or ground) at small feed rates.
- Change exit angle, use climb milling, lead-in/lead-out arcs.

## Scaling / thermal
- Use through-coolant and stable Vc to avoid thermal cracks on insert.
