# TiN — Oerlikon Balzers BALINIT A/B, Calico, Wikipedia
- BALINIT A ARCTIC (TiN): surface hardness 2300 HV, friction coefficient vs steel ~0.4
- Standard PVD TiN deposition 400–500°C
- Max operating / oxidation temp ~600°C (Balzers BALINIT A datasheet, widely cited)
- Typical thickness 1–4 µm, gold color
- Universal general-purpose coating for HSS & carbide; steels, stainless, cast iron
