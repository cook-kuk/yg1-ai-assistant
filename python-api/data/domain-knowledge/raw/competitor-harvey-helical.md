# Harvey Tool + Helical Solutions (Harvey Performance Company)

Sources: https://www.harveytool.com/, https://www.helicaltool.com/

## Helical Solutions - Variable Helix End Mills
- End mills for aluminum alloys (square, ball, corner radius) - typically 3-flute, Zplus/ZrN
- End mills for free-machining steels - 4/5-flute, AlTiN
- End mills for medium alloy steels - 4/5/7-flute
- End mills for hardened steels over 60 HRC - HEM optimized, 6-7 flute
- High-Efficiency Milling (HEM) optimized

## Harvey Tool - Miniature + Specialty End Mills
- Miniature end mills (long reach, square end) CNC ground from solid carbide
- Variable helix end mills for free-machining steels - square, ball, corner radius, finishers
- High-helix end mills for medium alloy steels - ball, tapered reach (mold cutters)
- Diameter down to 0.001"

Both brands: ISO N, P, M, H, S depending on line
Operations: HEM, slotting, profiling, mold finishing, micro-machining
