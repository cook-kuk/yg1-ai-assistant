# Machining Titanium — Kennametal (excerpt)
Source: https://www.kennametal.com/us/en/resources/blog/machining-titanium.html

Key points:
- Ti-6Al-4V thermal conductivity ~6.7 W/m·K (vs steel 50). Heat concentrates at the cutting edge.
- Use low cutting speeds (Vc 40–70 m/min for carbide), moderate feeds, high-pressure coolant (70+ bar ideal).
- Trochoidal / dynamic milling limits radial engagement (ae 8–15% D) while using full ap — spreads heat and prolongs tool life dramatically.
- Avoid dwelling — the tool work-hardens the surface; never stop feed while engaged.
- Sharp positive-rake, polished-flute carbide (TiAlN or AlTiN+) preferred; avoid built-up edge with flood or HPC.
- Rigid workholding essential; titanium's low modulus causes workpiece springback.
