# TiCN — Calico / BryCoat / Eifeler
- Hardness ~3000 HV (micro)
- Max oxidation temp ~400°C
- Friction coefficient 0.30–0.45 (vs steel ~0.2)
- Thickness 2–5 µm typical (3 µm common)
- Color: blue-grey / rose depending on C content
- Good for abrasive materials, cast iron, higher-feed milling
