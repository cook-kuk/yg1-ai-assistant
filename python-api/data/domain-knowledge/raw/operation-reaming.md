# Reaming — Sandvik Coromant

Source: https://www.sandvik.coromant.com/en-us/knowledge/reaming/troubleshooting-reaming

- Reaming corrects hole size and surface finish; removes 0.1–0.3 mm on diameter.
- Cutting speed: Vc for reaming ≈ 2/3 of drilling Vc in same material.
- Feed per revolution typically 2–3× drilling fn for the same diameter.
- Incorrect cutting data = the leading cause of reamer problems (chatter, bell-mouthing, oversize).
- Rigid toolholding or floating holder; flood or through-coolant required.
