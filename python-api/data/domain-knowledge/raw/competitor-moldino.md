# MOLDINO (ex-Mitsubishi Hitachi Tool)

Sources: http://www.moldino.com/en-US/products/endmill/epp-series/, https://data.moldino.com/catalog_pdf/eppth_e.pdf

## Epoch TH (EPP-TH series)
- Solid carbide end mill for high-hardness steels (up to 70 HRC)
- TH coating: high hardness + oxidation resistance, heat-resistant (dry cutting capable)
- Materials: H13, D2, A2 hardened dies
- Ball nose, corner radius, square
- Diameter 0.5-20 mm
- ISO H primary, P secondary

## EPP Series (Epoch Pencil Precision)
- Ball nose for roughing, semi-finishing, finishing
- Pre-hardened 35 HRC up to fully hardened
- Long-neck mold cutting

## Epoch SUS
- Dedicated stainless steel (ISO M) end mill
- Specialized geometry

## Epoch Micro (EMM / EMM-TH)
- Micro end mill, mold finishing
