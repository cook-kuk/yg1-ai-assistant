# Dormer Pramet Force X / Shark Line

## Force X (Gen 2) Solid Carbide Drill
- Multi-application solid carbide drill
- Continuously Thinned Web (CTW) for reduced thrust
- Through-coolant + non-coolant variants
- 3xD, 5xD, 8xD lengths
- ISO P / M / K reliable performance
- TiAlN PVD coating
- Diameter 3-20 mm typical
- Competes directly with YG-1 DREAM DRILLS general series

## Force M Solid Carbide Drill
- Stainless-optimized solid carbide drill
- ISO M primary

## Shark Line Taps
- Material-specific taps, color ring coding on shank
- Spiral flute / spiral point / straight flute variants
- ISO P / M / K / N / S by color
- HSS-E / HSS-PM body
