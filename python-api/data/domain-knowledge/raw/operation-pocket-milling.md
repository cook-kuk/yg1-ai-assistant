# Pocket milling — Harvey Performance / Helical

Source: https://www.harveyperformance.com/in-the-loupe/types-tool-entry/
Source: https://www.harveyperformance.com/in-the-loupe/ramping-success/

Key points:
- Helical (circular) ramp interpolation preferred to straight plunge for closed pockets/cavities.
- Programmed helix diameter should be >110–120% of the cutter diameter.
- Corner radius end mills during pocketing reduce corner breakdown and wear.
- HEM strategy: ae = 7–30% D, ap up to 2×D, with increased fz.
- Use 3–4 flute square/corner-radius end mill with internal coolant; avoid long-reach when not needed (use necked tools).
