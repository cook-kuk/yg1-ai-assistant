# Nachi Fujikoshi Carbide Lineup

Sources: https://www.nachiamerica.com/cutting-tools/products/, https://www.nachiamerica.com/products/148-aqua-mill-hard-end-mills/

## Aqua Drill EX (AQDEX)
- Solid carbide drill, Aqua EX coating (AlTiN)
- Flat type (AQDEXZR), Oil-hole type (through coolant)
- 10xD, 15xD, 20xD, 25xD, 30xD, 40xD lengths
- Diameter 0.2-20 mm
- 140 deg point angle, dual guiding chamfers
- ISO P / M / K general

## Aqua Mill Hard
- Hardened steel end mill, ball/square/corner radius
- Up to 65 HRC
- ISO H

## AG-Mill Roughing
- Roughing end mill with serrated (rougher) edge
- AG coating (AlCrN family)
- ISO P / K steel and cast iron roughing

## AG Power Drill
- Solid carbide drill, AG coating
