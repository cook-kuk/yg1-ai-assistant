# ISO 1832 — Indexable Insert Designation

Source: https://www.carbidedepot.com/formulas-insert-d.htm (and https://cdn.standards.iteh.ai/samples/69202/.../ISO-1832-2017.pdf)
Fetched: 2026-04-09

The ISO 1832 designation for an indexable insert is a string of up to 10 symbols. Positions 1-7 are mandatory; 8-10 are optional (or mandatory for PCBN/PCD).

## Position 1 — Shape
- H = hexagon (120°)
- O = octagon
- P = pentagon
- S = square (90°)
- T = triangle (60°)
- C = rhombic 80°
- D = rhombic 55°
- E = rhombic 75°
- M = rhombic 86°
- V = rhombic 35°
- W = trigon (80°)
- L = rectangular
- A = parallelogram 85°
- B = parallelogram 82°
- K = parallelogram 55°
- R = round

## Position 2 — Clearance (relief) angle
- A = 3°
- B = 5°
- C = 7°
- D = 15°
- E = 20°
- F = 25°
- G = 30°
- N = 0° (negative insert)
- P = 11°
- O = other

## Position 3 — Tolerance class
Tolerances on inscribed circle (d), thickness (s), and corner-point dimension (m).
- A, F, C, H, E, G, J, K, L, M, N, U  (each a specific tolerance row in ISO 1832)
- Commonly: C = close/precision, M = medium, U = utility/ordinary, G = ground top/bottom.

## Position 4 — Type / fixing and chip-breaker
- A = with hole
- B = with hole + one countersink
- C = with hole + two countersinks
- F = clamp-on, no hole, no chipbreaker
- G = with hole + chipbreakers both faces
- H = with hole + one countersink + chipbreaker one face
- J = with hole + two countersinks + chipbreaker both faces
- M = with hole + chipbreaker one face
- N = no hole, no chipbreaker
- Q = with hole + two countersinks
- R = no hole + chipbreaker one face
- T = with hole + one countersink + chipbreaker one face
- U = no hole + chipbreaker both faces
- W = with hole + one countersink
- X = special

## Position 5 — Cutting edge length (size)
Numeric, length of cutting edge in mm (rounded). Single- or two-digit (e.g., 12 = 12 mm).

## Position 6 — Thickness
Numeric, insert thickness in mm (e.g., 04 = 4.76 mm, 03 = 3.18 mm).

## Position 7 — Corner / nose radius
Two-digit, radius in 1/10 mm (e.g., 08 = 0.8 mm, 12 = 1.2 mm). 00 = sharp.

## Position 8 — Cutting edge condition
- F = sharp edge
- E = rounded (honed) edge
- T = chamfered (T-land)
- S = chamfered + honed
- K = double chamfered

## Position 9 — Cutting direction / hand
- R = right-hand
- L = left-hand
- N = neutral (no hand)

## Position 10 — Manufacturer option (chipbreaker / grade)
Free field, manufacturer's chipbreaker or geometry code (e.g., -MM, -PR, -QM).
