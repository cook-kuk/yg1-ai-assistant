# Kennametal HARVI Series Solid Carbide End Mills

Source: https://www.kennametal.com/us/en/products/metalworking-tools/milling/solid-end-milling/high-performance-solid-carbide-end-mills/

## HARVI III
- 6-flute unequal flute spacing (chatter-free), eccentric relief grind
- KCSM15 Beyond PVD coating
- For traditional + high-velocity / HPC machining
- Square end, ball nose, corner radius variants; inch + metric
- Typical diameter range 3-25 mm
- ISO P / M / S (steel, stainless, high-temp alloys)

## HARVI I TE
- 4-flute asymmetrical divided flutes, variable helix, twisted end face, eccentric relief, chip gashes
- Roughing AND finishing with one tool
- ISO P / M / K / S
- Ball nose, square end; 3-20 mm
- TiAlN/AlCrN (KCPM15/KCSM15)

## HARVI I
- General-purpose high-performance end mill
- Unequal flute spacing to minimize chatter

Operations: slotting, shoulder milling, roughing, finishing, ramping
