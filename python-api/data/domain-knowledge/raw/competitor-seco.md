# Seco Jabro Solid Carbide

Sources: https://www.secotools.com/article/67271, https://ctemag.com/products/jabro-solid2-js554-solid-carbide-endmills/

## JS554 (Jabro-Solid2)
- 3-4 flute high-performance square end mill
- Diameter 3-20 mm, LOC 2xD+2mm
- Corner radius + 45 deg chamfer versions
- Cylindrical + Weldon shanks
- NXT PVD coating (SIRON-A family)
- +30% tool life, +20% speed vs JS553
- ISO P / M / K (universal)

## JS553
- Predecessor to JS554
- 3-flute universal

## JC880 (Jabro Composites)
- Single end mill center-cutting low-helix
- CVD diamond coating for CFRP/GFRP/PEEK/honeycomb
- 8 mm x 0.2 mm CR x 24 mm LOC typical

## JHP (Jabro HPM - High Performance Milling)
- High-performance series, multi-flute roughing/finishing

Operations: slotting, shoulder, profile, composite contouring
