# Side milling / profiling — Harvey Performance, Sandvik

Source: https://www.harveyperformance.com/in-the-loupe/depth-of-cut/
Source: https://www.sandvik.coromant.com/en-gb/knowledge/milling/profile-milling

- Finishing peripheral cut: ae ≈ 3–5% of cutter diameter per radial pass.
- Heavy roughing: ae ≈ 30–50% of cutter diameter.
- Climb (down) milling preferred for surface finish and tool life.
- 4–5 flute square/corner-radius tool with 35–45° helix; 50°+ for chatter-prone thin walls.
- ap up to 1–2×D with long-edge tools when rigidity allows.
