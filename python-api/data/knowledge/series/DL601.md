# AIRCRAFT DRILLS — DL601
**제품명**: HSS Co5, AIRCRAFT EXTENSION DRILL 135° SPLIT POINT

## 기본 사양
- 공구 타입: Drill
- 서브타입: Extension Drill
- 공구 재질: HSS Co5
- 코팅: Gold Coloring
- 헬릭스각: 30°

- Right hand spiral
- 135° split point
- Improved chip removal in most materials
- Especially suitable for deep drilling applications
- NAS 907
- Recommended cutting conditions page for aircraft drills

## 용도
- Most materials
- Deep drilling applications
- Aircraft drilling

## 상세 정보 (원문 요약)
- [page_0361] HSS Co5 AIRCRAFT EXTENSION DRILL 135° SPLIT POINT 시리즈이다. Right hand spiral, 30° helix, 135° split point이며, 대부분의 소재에서 칩 배출이 향상되고 특히 deep drilling applications에 적합하다. Fractional size로 5/64"부터 1/2"까지 제공되며, 6"와 12" overall length 규격이 있다.
- [page_0368] AIRCRAFT DRILLS RECOMMENDED CUTTING CONDITIONS 페이지로, DL601 시리즈의 추천 절삭조건을 제시한다. 본 페이지는 DL601, DL602, DL603, D1631, D1632, D1633 및 DL604, DL605, DL606, D1634, D1635, D1636 시리즈의 절삭조건 표를 포함하지만, 제품의 상세 사양은 없고 헤더 수준의 정보만 제공한다.
