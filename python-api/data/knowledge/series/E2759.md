# YG-1 — E2759
**제품명**: MULTI FLUTE ROUGHING - SIDE CUTTING / 多刃粗加工-侧铣削

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Multi Flute Roughing
- 공구 재질: M42 HSS
- 직경 범위: D10.0~D50.0
- 깊이비: 0.5D / 1.5D

## 대응 ISO 그룹
P, N

## 대상 피삭재
- Non-alloy steel
- Low alloy steel
- High alloyed steel
- Tool steel
- Aluminum-wrought alloy
- Aluminum-cast alloyed
- 탄소강
- 합금강
- 공구강
- 알루미늄
- 알루미늄 합금

## 특징
- Side cutting roughing end mill
- Cutting condition table provided by ISO material group and diameter
- Recommended Ae = 0.5D and Ap = 1.5D
- For long & extra long types, feed should be reduced by around 50%
- M42 HSS
- 표준 / 짧은길이 / 롱 / 엑스트라 롱 라핑

## 용도
- Roughing side milling

## 상세 정보 (원문 요약)
- [page_1483] E2759 SERIES MULTI FLUTE ROUGHING - SIDE CUTTING / 多刃粗加工-侧铣削의 절삭조건 페이지입니다. ISO P와 N 재질군에 대해 직경 10.0~50.0mm의 Vc, fz, RPM, FEED 추천값이 표로 제시되어 있으며, Ae는 0.5D, Ap는 1.5D로 안내됩니다. 장날 및 장경 타입은 FEED를 약 50% 줄여야 한다는 주의사항이 있습니다.
- [page_0657] M42 HSS 표준 / 짧은길이 / 롱 / 엑스트라 롱 라핑 시리즈의 추천 절삭조건 페이지이다. E2758 / E2759 / E2760 / E2751 / E2714 / E2752 시리즈에 대해 탄소강·합금강·공구강(P)과 알루미늄/알루미늄 합금(N) 가공 조건이 제시되어 있다. 절삭조건 표에는 직경 6.0~50.0에 대한 Vc, Fz, RPM, Feed가 수록되어 있으며 Ap=1.5D, Ae=0.1D로 안내된다.
