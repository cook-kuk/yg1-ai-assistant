- [page_0397] STRAIGHT SHANK DRILLS / DL510, DL508, DL509, DL505, DL504, DL608 SERIES. HSS-E, DH100 WORM PATTERN DRILLS로 표시된 추천 절삭조건 페이지이며, 본문은 ISO P/K 재질별 절삭조건 표로 구성되어 있습니다. 직선 생크 드릴의 권장 Vc, RPM, FEED가 지름 D2.0~D30.0 범위로 제시됩니다.
- [page_0156] DL504 SERIES는 HSS-E, STRAIGHT SHANK TWIST DRILLS for DEEP HOLES - LONG 제품이다. Steam Tempered(Black Oxide Finish) 표면처리이며, DIN 340, 38°, h8, 130°, Vap, DH100 worm pattern drills 아이콘 정보가 표시된다. 비합금강, 합금강, 회주철, 가단주철, 특수 알루미늄/마그네슘 합금의 심공 가공에 적용되며, TiN(DN504), TiCN(DX504), TiAIN(DT504) 코팅이 요청 시 가능하다.
