# YG-1 — E2051
**제품명**: HSS Co8 & HSS, 4 FLUTE REGULAR LENGTH DOUBLE

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Double End / Regular Length / Flat
- 날 수: 4
- 공구 재질: HSS Co8 (M42) / HSS (M2)
- 코팅: Uncoated; optional TiN, TiCN, TiAlN, Hardslick
- 헬릭스각: 30 degree
- 생크: Flat shank; shank same diameter as cutting portion
- 직경 범위: 1/2 to 1

## 대응 ISO 그룹
P, M, K, N, S, H

## 대상 피삭재
- Non-alloy steel
- Low alloy steel
- High alloyed steel
- Tool steel
- Stainless steel
- Grey cast iron
- Nodular cast iron
- Malleable cast iron
- Aluminum-wrought alloy
- Aluminum-cast alloyed
- Copper and Copper Alloys (Bronze / Brass)
- Non Metallic Materials
- Heat Resistant Super Alloys
- Titanium Alloys
- Hardened steel
- Chilled Cast Iron
- Hardened Cast Iron

## 특징
- Double-end version of E2031 four flute tools
- Used for the same type of finishing operation
- Two tools on one shank saves on sharpening set-up and initial tool costs
- Easy to regrind
- Available in 8% cobalt (M42) and HSS (M2)
- Uncoated standard; coated versions available on request

## 용도
- Finishing operation

## 상세 정보 (원문 요약)
- [page_1781] E2051는 HSS Co8 & HSS 4날 레귤러 길이 더블 엔드밀 시리즈로, E2031 4날 공구의 더블엔드 버전이며 동일한 마무리 가공에 사용된다. 한 개의 샹크에 두 개의 공구를 사용해 연삭 셋업과 초기 공구 비용을 줄일 수 있고, 재연삭이 쉽다. 8% 코발트(M42)와 HSS(M2) 재질로 제공되며, 기본은 무코팅이고 TiN, TiCN, TiAlN, Hardslick 코팅이 요청 시 가능하다.
