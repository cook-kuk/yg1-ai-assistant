# E-FORCE — GNX35
**제품명**: 2날 스쿼어 GNX35 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Square
- 날 수: 2
- 직경 범위: D0.2~D12.0
- 깊이비: 1.0D

## 대응 ISO 그룹
P, H

## 대상 피삭재
- 프리하든강
- 고경도강

## 특징
- 추천 절삭조건 페이지
- 슬로팅 조건 수록
- Ap 0.05D, Ae 1.0D 조건
- 사이드컷팅

## 용도
- 슬로팅

## 상세 정보 (원문 요약)
- [page_0111] 추천 절삭조건 E-FORCE / 2날 스쿼어 GNX35 시리즈의 슬로팅 절삭조건이 수록되어 있다. 프리하든강(P, HRC 35~45)과 고경도강(H, HRC 45~70)에 대해 인선(mm)별 Vc, Fz, RPM, Feed 값이 제시된다. 모든 조건에서 Ap는 0.05D, Ae는 1.0D로 표기되어 있다.
- [page_0112] 추천 절삭조건 E-FORCE의 2날 스퀘어 GNX35 시리즈 사이드컷팅 절삭조건 페이지이다. 프리하든강(P, 35~45/45~55HRC)과 고경도강(H, 55~65/65~70HRC)에 대한 Vc, Fz, RPM, Feed 조건이 D1.0~D12.0까지 제시되어 있다. Ap는 1.0D, Ae는 0.03D로 표기되어 있다.
