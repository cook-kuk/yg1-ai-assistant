# YG-1 — CD2101
**제품명**: 코발트 드릴 D2101 / CD2101 / D2104 / D2107 / DL109 / D2105 / D2201 / DL205 시리즈

## 기본 사양
- 공구 타입: Drill
- 서브타입: Cobalt Drill
- 공구 재질: Cobalt
- 직경 범위: D1.0~D8.0

## 대응 ISO 그룹
P, M, K, N, S

## 상세 정보 (원문 요약)
- [page_0944] 추천 절삭조건 페이지로, 코발트 드릴 D2101 / CD2101 / D2104 / D2107 / DL109 / D2105 / D2201 / DL205 시리즈의 절삭조건을 ISO P, M, K, N, S 재질별로 제시한다. 드릴직경 1.0, 2.0, 3.0, 4.0, 6.0, 8.0에 대해 Vc, RPM, Feed 범위를 제공하며, 저탄소강, 중탄소강, 고탄소강, 스테인레스강, 주철, 알루미늄, 티타늄 합금 등 적용 재질이 포함된다.
