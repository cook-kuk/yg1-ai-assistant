# YG-1 — G9540
**제품명**: CARBIDE, 4 FLUTE LONG LENGTH

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Long Length
- 날 수: 4
- 공구 재질: CARBIDE
- 코팅: TiAlN
- 헬릭스각: ≈ 30°
- 생크: DIN 6535HA
- 직경 범위: D3.5~D20.0

## 대응 ISO 그룹
P, M, K, N, S, H

## 특징
- Suitable for dry milling applications at high temperatures.
- Excellent high-performance end mills.
- 4 flute allows for better work piece finishes.
- Für die Trockenbearbeitung.
- Hervorragendes Preis - Leistungsverhältnis.
- 4 Schneiden erzeugen eine bessere Oberflächengüte des Werkstücks.

## 용도
- Dry milling applications at high temperatures

## 상세 정보 (원문 요약)
- [page_1627] CARBIDE, 4 FLUTE LONG LENGTH 시리즈 G9540은 건식 고온 밀링용 고성능 엔드밀이다. 4날 구조로 가공면 품질이 좋고, TiAlN 코팅과 DIN 6535HA 샹크를 적용했다. 직경은 3.5mm부터 20.0mm까지이며, P/M/K/N/S/H 계열 재질에 폭넓게 추천된다.
