# V7 PLUS — GMH60
**제품명**: 4날 스퀘어 / 래디우스 GMH60 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Square / Radius
- 날 수: 4
- 직경 범위: 3.0~25.0

## 대응 ISO 그룹
P, K, M, S

- 합금강
- 공구강
- 주철
- 스테인레스강 300
- 스테인레스강 400
- 스테인레스강(PH)
- 티타늄
- 내열합금

- 슬로팅용 추천 절삭조건
- 사이드컷팅

## 용도
- 슬로팅

## 상세 정보 (원문 요약)
- [page_0514] 추천 절삭조건 V7 PLUS의 4날 스퀘어 / 래디우스 GMH60 시리즈 슬로팅 조건 페이지이다. ISO P/K/M/S 소재별로 Vc, Fz, RPM, Feed가 인선 3.0~25.0mm에 대해 제시되어 있으며, Ap와 Ae는 대부분 1.0D(티타늄/내열합금은 Ap 0.5D, Ae 1.0D)로 표기되어 있다.
- [page_0515] 추천 절삭조건 V7 PLUS 페이지로, 4날 스퀘어/래디우스 GMG86 / GMH60 / GMG87 시리즈의 사이드컷팅 조건을 제시한다. 탄소강, 합금강, 공구강, 주철, 스테인레스강 300/400/PH, 티타늄, 내열합금에 대해 ISO별 Vc, Fz, RPM, Feed가 표로 정리되어 있다. 절삭조건은 인선 3.0~25.0mm 범위에 대해 제공되며, Ap/Ae는 재질별로 1.5D 또는 1.0D, 0.5D 또는 0.35D/0.25D로 안내된다.
