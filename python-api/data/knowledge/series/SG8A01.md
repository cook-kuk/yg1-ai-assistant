## 상세 정보 (원문 요약)
- [page_0353] 추천 절삭조건 X5070 S / 2날 스퀘어 SG8A01 시리즈의 슬로팅 절삭조건 페이지이다. ISO P(합금강, 내열강)와 ISO H(40~70HRc 고경도강)용으로 Vc, Fz, RPM, Feed가 인선(mm)별로 제시되어 있다. Ap는 0.05D 또는 0.02D, Ae는 1.0D로 안내된다.
