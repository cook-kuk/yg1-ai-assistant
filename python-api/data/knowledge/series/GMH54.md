# YG-1 — GMH54

- 공구 타입: End Mill
- 서브타입: Square

- 직경 범위: 4.0~25.0

- 4날 스퀘어 엔드밀 시리즈

- 슬로팅

## 상세 정보 (원문 요약)
- [page_0533] 추천 절삭조건 페이지로, 4날 스퀘어 GMH54 시리즈의 사이드컷팅 조건을 제시한다. 탄소강, 합금강, 공구강, 주철, 스테인레스강(300/400/PH), 티타늄, 내열합금에 대한 Vc, Fz, RPM, Feed, Ap, Ae가 수록되어 있으며, 직경 4.0~25.0mm 기준으로 조건이 정리되어 있다. 본 페이지는 수치 절삭조건 중심이며, 최적 상황 기준 데이터로 장비 상태와 가공 환경에 따라 조정 권고 문구가 포함되어 있다.
- [page_0534] 추천 절삭조건 페이지로, 4날 스퀘어 GMH54 시리즈의 슬로팅 조건을 제시한다. 인선(mm) 4.0~25.0에 대해 탄소강, 합금강, 공구강, 주철, 스테인레스강 300/400/PH, 티타늄, 내열합금의 Vc, Fz, RPM, Feed, Ap, Ae가 표로 정리되어 있다. 본 페이지는 절삭조건 중심이며, 최적 조건은 장비 상태와 가공 환경에 따라 조절할 것을 안내한다.
