## 상세 정보 (원문 요약)
- [page_0833] TM933 SERIES는 ISO metric coarse threads DIN 13용 머신 탭이다. 항공우주 및 화학 산업에서 사용되는 니켈 합금과 내열 합금강의 탭핑에 적합하며, 홀 타입은 2.5×D이다. DIN 371은 M2~M10, DIN 376는 M11~M30에 적용되며, 규격은 M2부터 M30까지 구성된다. 적용 재질 표에는 니켈계/내열계 재질뿐 아니라 티타늄 합금, 경화강, 칠드 주철 등도 표시되어 있다.
- [page_0354] ISO metric coarse threads DIN 13용 머신 탭 시리즈로, 니켈 합금과 내열 합금강의 탭핑에 적합하다. HSS PM 재질이며 Bright 코팅, 6H 공차, 60° 형상, R40 사양으로 제공된다. DIN 371(M2~M10) 및 DIN 376(M11~M30) 규격으로 구성되어 있고, 홀 타입은 2.5×D이다.
