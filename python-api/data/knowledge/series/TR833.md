## 상세 정보 (원문 요약)
- [page_0838] TR833 시리즈는 YG TAP Ti Ni의 ISO metric coarse threads DIN 13용 머신 탭이다. 특수한 홈 형상으로 절삭칩 배출이 우수하여 막힌 구멍(Blind hole) 가공에 적합하며, Hole type은 2.5xD이다. HSS PM 재질, Bright coating, 6H 공차, 60° 형상, R40 헬릭스 각을 가지며 DIN 371(M2~M10), DIN 376(M12) 규격으로 제공된다.
- [page_0356] ISO metric coarse threads DIN 13용 머신 탭 시리즈로, 블라인드 홀 가공에 적합하도록 특수한 홈 형상과 우수한 칩 배출성을 갖는다. DIN 371(M2~M10) 및 DIN 376(M12) 규격이며, 홀 타입은 2.5×D이다. 적용성 표는 TQ833와 동일하며, 주로 P와 M 재질에 대해 추천된다.
