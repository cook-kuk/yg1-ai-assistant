# SYNCHRO TAP — TTS37
**제품명**: SYNCHRO TAP TTS37 SERIES / ISO Metric Coarse Threads DIN 13

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap / Synchro tap
- 공구 재질: HSS-PM
- 코팅: TiN (TiAlN available on request)
- 생크: DIN 371 / DIN 376
- 직경 범위: M3~M20
- 깊이비: 3.0xD

## 특징
- Coated HSS-PM (Powder Metallurgy) taps for high-speed tapping on rigid CNC machines or equivalent machines
- Up to 3 times faster in tapping compared to conventional taps
- For high-speed synchro tapping, synchro holder for increasing tool life and thread quality is recommended
- High precision threads
- Hole type: 3.0xD

## 용도
- ISO Metric Coarse Threads DIN 13
- Machine tapping on rigid CNC machines or equivalent machines
- High-speed synchro tapping
