## 실적 / 케이스 스터디
- X-POWER PRO and X-Power showed the same milling length in three tests: 100m for 2-flute square end mills, 120m for 2-flute ball end mills, and 100m for 4-flute corner radius end mills. The tests were performed on KP4M(HRC35) / DIN 1.2311, ANSI P20+Ni with oil mist coolant.
- X-POWER PRO end mills were tested against X-Power on KP4M(HRc35) / DIN 1.2311 / ANSI P20+Ni with oil mist coolant. The page reports 67% better wear resistance for the 2-flute square end mill, 80% better for the 2-flute ball end mill, and 50% better for the 4-flute corner radius end mill.
- X-POWER PRO end mills were tested against X-Power in three cutting tests: 2 flute square end mills, 2 flute ball end mills, and 4 flute corner radius end mills. The page reports 67%, 80%, and 50% better wear resistance respectively, all in KP4M(HRc35) / DIN 1.2311 / ANSI P20+Ni with oil mist coolant.
