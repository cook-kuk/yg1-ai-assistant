# YG-1 — VBMT-VBGT
**제품명**: VBMT- VBGT [35° 포지티브]

## 기본 사양
- 공구 타입: Insert
- 서브타입: Turning insert

## 대응 ISO 그룹
P, M, K, N, S, H

## 대상 피삭재
- 일반강
- SUS강
- 주철
- 비철합금
- 내열합금
- 고경도

## 특징
- 35° 포지티브 터닝 인서트
- 칩브레이커 UF, UG, MF, MM, SF 구성
- 재고관리 표시가 있는 품목

## 용도
- 터닝
- 정삭용
- 중삭용
- SUS 가공용 정삭/중삭

## 상세 정보 (원문 요약)
- [page_1470] VBMT- VBGT [35° 포지티브] 터닝 인서트 페이지로, 칩브레이커별 형번과 권장 이송량/절입량이 정리되어 있다. UF는 정삭용, UG는 중삭용, MF와 MM은 SUS용 정삭/중삭, SF는 정삭용으로 구분된다. 일반강(P), SUS강(M), 주철(K), 비철합금(N), 내열합금(S), 고경도(H) 대상 재질별 적용 범위가 표시되어 있다.
