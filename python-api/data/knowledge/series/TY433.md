# YG TAP ALU — TY433
**제품명**: ISO metric coarse threads DIN 13 / Machine taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 날 수: 3
- 공구 재질: HSS-E
- 코팅: TiAIN
- 선단각: 60°
- 생크: DIN 371 / 376
- 깊이비: 2.0xD

## 대응 ISO 그룹
P, M, N, S, H

## 대상 피삭재
- Brass
- Short chip work materials
- Non-alloy steel
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- Suitable for brass and short chip work materials
- DIN profile not ISO
- DIN 371(M2~M10) and DIN 376(M11~M30)

## 용도
- Gewindeschneiden von Messung und anderen kurzspanenden Werkstoffen

## 상세 정보 (원문 요약)
- [page_0365] ISO metric coarse threads DIN 13용 YG TAP ALU TY433 시리즈 머신 탭이다. HSS-E 재질에 TiAIN 코팅, 공차 6H, 각도 60°이며, 황동과 단칩성 소재에 적합하다. Hole type은 2.0xD이고 DIN 371/376 규격을 따른다.
- [page_0366] 추천 절삭조건 페이지로, YG TAP ALU의 재질별 권장 절삭조건을 제시한다. 상단 헤더에 TC163, TC963, TC169, TC170, TE953, TC622, TE943, TC433, TE443, TY433 시리즈가 표시되어 있으며, 본 페이지는 숫자 절삭조건 표 중심이다. 제품 상세 설명은 없고, ISO P/N 재질군에 대한 적용 재질만 확인된다.
