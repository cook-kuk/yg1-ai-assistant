# 4G Mills — SEMD98060E

- 공구 타입: End Mill
- 서브타입: Ball Nose
- 날 수: 2
- 냉각: Oil Mist
- 직경 범위: Ø6

H, P

## 대상 피삭재
- KP4M (HRc35 / DIN 1.2738 Improved)

- Case study page comparing YG-1 vs Competitor A
- Ball nose end mill test in profile cutting
- Wear vs. milling length graph
- Case study page for 4G Mills end mills

- Ball nose tool used for machining KP4M (HRc35 / DIN 1.2738 Improved)

## 용도
- Profile Cutting

## 실적 / 케이스 스터디
- Ball nose test on KP4M (HRc35 / DIN 1.2738 Improved) using 2Flute SEMD98060E. YG-1 and Competitor A were both tested to a total milling length of 300m under the same cutting conditions, with wear compared by graph.
- TEST I는 2날 볼 노즈 엔드밀 SEMD98060E를 사용한 Profile Cutting 사례로, KP4M(HRc35 / DIN 1.2738 Improved) 소재에서 YG-1 4G Mills와 CompetitorA를 동일 조건으로 비교했다. 두 공구 모두 총 밀링 길이 300m까지 테스트되었으며, 절삭 조건과 공구 크기, 냉각 방식이 함께 제시된다.

## 상세 정보 (원문 요약)
- [page_0056] TEST I는 Ball Nose(球头) 4G Mills End mills의 사례 분석 페이지로, YG-1과 Competitor A의 마모(wear)를 밀링 길이에 따라 비교한 그래프가 제시되어 있다. 공구는 2날 SEMD98060E, 사이즈 Ø6×6×12×90이며, 가공재는 KP4M(HRc35 / DIN 1.2738 Improved), 가공은 Profile Cutting 조건에서 수행되었다. 절삭조건은 Vc 130.061 m/min, 6,900 rpm, Feed 830 mm/min, fz 0.060 mm/tooth, 축방향 0.2 mm / 반경방향 1.2 mm, 냉각은 Oil Mist, 오버행 26 mm, 장비는 Machining Center이다.
- [page_0064] TEST I는 2날 볼 노즈 엔드밀 SEMD98060E를 사용한 Profile Cutting 사례로, KP4M(HRc35 / DIN 1.2738 Improved) 소재에서 YG-1 4G Mills와 CompetitorA를 동일 조건으로 비교했다. 두 공구 모두 총 밀링 길이 300m까지 테스트되었으며, 절삭 조건과 공구 크기, 냉각 방식이 함께 제시된다.
