# YG TAP STEEL — TR823
**제품명**: ISO metric coarse threads DIN 13 / Machine taps / Maschinengewindebohrer

- 깊이비: 2.5×D

- Excellent chip evacuation
- DIN 371 / DIN 376
- 2.5×D hole type
- Recommended tool holders: Plain Shank, TAPPING ER CHUCK, TAPPING CHUCK, ONE STEP TAPPING CHUCK

## 용도

- Blind hole tapping

## 상세 정보 (원문 요약)
- [page_0734] TR823 시리즈는 YG TAP STEEL의 ISO metric coarse threads DIN 13용 머신 탭이다. 특수한 홈 형상으로 절삭유 배출이 우수하여 특히 막힌 구멍 가공에 적합하며, 2.5×D 홀 타입과 DIN 371 / DIN 376 규격을 따른다. M2~M10은 DIN 371, M12는 DIN 376이며, Plain Shank 및 다양한 탭핑 척 사용이 권장된다.
