# YG-1 — ENMX09
**제품명**: ENMX 가공 데이터

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Face Mill / Shell / Shank
- 생크: 생크, 쉘 (Face Mill)
- 직경 범위: D25~D80

## 특징
- 가공 방식별 추천 절삭조건 제공
- 경사 가공, 플런징 가공, 헬리컬 램핑 가공, 확장 가공 조건 수록
- 홀더 타입별로 생크와 쉘(Face Mill) 구분

## 용도
- 일반 가공
- 램핑 가공
- 플런징 가공
- 프로파일링 가공
- 헬리컬 가공
- 확장 가공

## 상세 정보 (원문 요약)
- [page_1545] ENMX 가공 데이터 페이지로, ENMX09 시리즈의 가공 방식별 추천 절삭조건을 정리한 표이다. 생크와 쉘(Face Mill) 홀더 타입별로 가공경, 최대 절입 APMX, 경사각, 플런징 폭, 헬리컬 램핑 가공의 최소/최대 가공경과 피치, 확장 폭을 제시한다. ENMX09는 D25~D80 범위를 포함한다.
