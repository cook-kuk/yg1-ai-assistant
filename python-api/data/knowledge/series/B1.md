## 상세 정보 (원문 요약)
- [page_0541] YG TAP INOX 선택 가이드 페이지로, Spiral Flute 탭의 재질/코팅/시리즈 매칭을 보여준다. B1은 HSS-E, Bright, R45 스파이럴 플루트 조건으로 UNC/UNF에 대응한다. 아래 표는 ISO 재질군별로 ◎/○ 등급과 VDI 3323 경도 범위를 제시한다.
