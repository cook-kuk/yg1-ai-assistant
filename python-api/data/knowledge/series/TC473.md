## 상세 정보 (원문 요약)
- [page_0910] TC473 시리즈는 YG TAP GENERAL의 MF ISO metric fine threads DIN 13용 머신 탭이다. DIN 374 규격이며, 얕은 구멍 가공에 적합하고 Hole type은 2.0×D이다. 재질은 HSS-E, 6H 공차, 60° 형상, Bright 마감이며 Plain Shank와 TAPPING ER CHUCK / TAPPING CHUCK / ONE STEP TAPPING CHUCK 사용 아이콘이 표시되어 있다. M4×0.5부터 M24×1.5까지 다양한 규격이 수록되어 있고, ISO P/M/K/N/S/H 재질군에 대한 추천도가 함께 제시된다.
- [page_0391] TC473 SERIES는 ISO metric fine threads DIN 13용 머신 탭이다. 얕은 홀 가공에 적합하며 Hole type은 2.0xD이다. DIN 374 규격으로 제공되며, 표에는 M4부터 M16까지의 다양한 피치 조합과 전체 길이, 넥 길이, 샹크 직경, 스퀘어 치수, 플루트 수, 드릴 직경이 정리되어 있다.
