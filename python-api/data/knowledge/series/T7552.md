# YG-1 — T7552
**제품명**: PS STRAIGHT PIPE TAPS for PS THREADS

## 기본 사양
- 공구 타입: Tap
- 서브타입: Straight Pipe
- 날 수: 4, 5, 6
- 공구 재질: HSS
- 생크: Square shank
- 직경 범위: PS 1/8~PS 2"
- 깊이비: 2.0×D

## 대응 ISO 그룹
P, M, K, N, S, H

## 특징
- Suitable for process Straight pipe internal threads
- PS: Straight pipe threads (for pressure-tight joints)
- Hole type: 2.0×D
- JIS II limit

## 용도
- Straight pipe internal thread tapping
- Pressure-tight joint thread machining

## 상세 정보 (원문 요약)
- [page_0791] T7552 SERIES / T2552 SERIES는 PS THREADS용 직관 파이프 탭이다. PS(pressure-tight joints) 직관 내나사 가공에 적합하며, Hole type은 2.0×D이다. 규격은 PS 1/8부터 PS 2"까지 있으며, HSS와 HSS-E EDP No.가 함께 제시되고, 플루트 수는 규격에 따라 4, 5, 6으로 구성된다.
