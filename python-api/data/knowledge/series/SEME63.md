# 4G MILLS — SEME63
**제품명**: 3날 고이송용 래디우스(더블코너 래디우스) SEME63 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Radius / Double Corner Radius / High Feed
- 날 수: 3

## 대응 ISO 그룹
P, H

## 대상 피삭재
- 합금강
- 탄소강
- 프리하든강
- 고경도강

## 특징
- 3날 고이송용 래디우스(더블코너 래디우스)
- 일반가공 및 고속가공 절삭조건 제공
- 인선(mm) 6.0, 8.0, 10.0, 12.0, 16.0, 20.0 기준 조건 수록

## 용도
- 일반가공
- 고속가공

## 상세 정보 (원문 요약)
- [page_0281] 3날 고이송용 래디우스(더블코너 래디우스) SEME63 시리즈의 추천 절삭조건 페이지이다. 일반가공과 고속가공 조건이 각각 제시되어 있으며, 합금강/탄소강(P ~35), 프리하든강(P 35~45), 고경도강(H 45~55)에 대한 Vc, Fz, RPM, Feed 값이 인선 6.0~20.0mm 기준으로 정리되어 있다. Ap는 주로 0.03D, Ae는 일반가공에서 0.5D, 고경도강에서는 0.3D로 제시된다.
