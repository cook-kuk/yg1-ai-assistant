## 상세 정보 (원문 요약)
- [page_1520] CARBIDE 6날 멀티플 렝스 엔드밀 V7 PLUS A 시리즈로, variable pitch 형상으로 chatter-free 절삭과 고속/트로코이달 밀링에 적합하다. Stainless steel, mild steel, cast iron 및 HRc40 이하의 저/중경도 소재에 우수한 성능을 보이며, reinforced cutting edge가 특징이다. Square는 UGMG21 시리즈, corner radius는 UGMG23 시리즈로 구분된다.
- [page_1531] V7 PLUS의 UGMG21 시리즈에 대한 추천 절삭조건 페이지로, 6날 사이드 커팅용 조건이 제시되어 있다. ISO P/M/K/S 재질군별로 SFM(Vc), IPT(fz), RPM, IPM(Feed) 값이 표로 정리되어 있으며, 절삭 길이가 2D 미만일 경우 L.O.C x 90%를 적용하라는 주의사항이 있다.
