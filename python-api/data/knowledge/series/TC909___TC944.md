# YG-1 — TC909 / TC944
**제품명**: SCREW THREAD INSERT TAP RECOMMENDED CUTTING CONDITIONS / HSS-E NUT TAP

## 기본 사양
- 공구 타입: Tap
- 서브타입: Screw Thread Insert Tap / Nut Tap
- 공구 재질: HSS-E

## 대응 ISO 그룹
P, N

## 대상 피삭재
- Non-alloy steel
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- Recommended cutting conditions are provided for screw thread insert taps
- HSS-E nut tap for nut tapping machines

## 용도
- Zum Gewindeschneiden von Muttern
- Nut Tapping Machines

## 상세 정보 (원문 요약)
- [page_0408] SCREW THREAD INSERT TAP RECOMMENDED CUTTING CONDITIONS / EMPFOHLENE SCHNEIDKONDITIONEN 페이지이며, 하단에 HSS-E NUT TAP / MUTTERGEWINDEBOHRER가 표시되어 있습니다. TC909 / TC944에 대한 추천 절삭조건이 ISO P, N 재질별로 제시되며, 비합금강, 알루미늄(단조/주조), 구리 및 구리합금에 대한 HB/HRc 기준과 절삭조건 범위가 포함되어 있습니다. 또한 Nut Tapping Machines용, Muttern의 나사 가공용 제품임을 알 수 있습니다.
