## 상세 정보 (원문 요약)
- [page_0689] Vap TB804 시리즈는 ISO Metric coarse threads DIN 13용 머신 탭이며, HSS-E 재질에 R40 헬릭스, 60° 각도, 6H 공차, C 챔퍼 사양이다. DIN 371/376 규격으로 M2~M22까지 제공되며, 3날(표기상 No. of Flute 3) 구조와 2.5×D 홀 타입을 가진다. 비합금강, 저합금강, 고합금강/공구강, 스테인리스, 주철, 알루미늄, 구리합금, 내열합금, 티타늄합금 등 폭넓은 재질에 적용된다.
- [page_0690] ISO Metric coarse threads DIN 13용 COMBO TAP의 Vap 코팅 시리즈이다. HSS-E 재질의 머신 탭이며 DIN 371/376, 6H, 60° 사양으로 2.5×D 홀 타입에 사용한다. 특수 절삭 형상으로 범용 사용, 정확한 나사 형상, 긴 공구 수명을 강조하며, Plain Shank 및 다양한 탭 척 사용을 권장한다.
- [page_0299] TB804는 Combo Spiral Flute Tap Set으로, VAP 표면처리의 세트 구성품이다. M5, M6, M8, M10, M12 규격이 5pcs로 제공된다.
