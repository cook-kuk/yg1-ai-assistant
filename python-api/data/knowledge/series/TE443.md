# YG TAP ALU — TE443
**제품명**: YG TAP ALU - TE443 SERIES / ISO metric coarse threads DIN 13 / Machine taps

- 서브타입: Machine tap
- 날 수: 3-4
- 공구 재질: HSS-E
- 코팅: Nitride
- 선단각: 60°
- 생크: Plain Shank
- 직경 범위: M2~M30
- 깊이비: 2.0×D

## 대상 피삭재

- Short chip work materials
- Non-alloy steel
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- Suitable for brass and short chip work materials
- DIN 371 / DIN 376
- Hole type: 2.0×D
- 6HX thread form
- 60° thread angle
- Recommended tool holders: Plain Shank, TAPPING ER CHUCK, TAPPING CHUCK, ONE STEP TAPPING CHUCK
- DIN profile not ISO
- DIN 371(M2~M10) and DIN 376(M11~M30)

## 용도
- Thread cutting in brass and other short-chip materials
- Gewindeschneiden von Messung und anderen kurzspanenden Werkstoffen
