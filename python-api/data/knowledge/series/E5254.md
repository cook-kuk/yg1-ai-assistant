# BANSHEE — E5254
**제품명**: SOLID CARBIDE & HSS-PM ALU POWER END MILLS

- 공구 타입: End Mill
- 서브타입: Square
- 날 수: 2
- 공구 재질: Solid Carbide / HSS-PM
- 코팅: TiCN
- 헬릭스각: 42°
- 생크: Plain Shank
- 직경 범위: D1/16~D1

The page marks it as BANSHEE and indicates it is suitable for ISO N materials, especially aluminum wrought/cast alloys.
