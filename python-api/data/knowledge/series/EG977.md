# ALU-POWER — EG977
**제품명**: ALU-POWER END MILLS EG977 SERIES TiCN Coated

## 기본 사양
- 공구 타입: End Mill
- 서브타입: 3 Flute
- 날 수: 3
- 코팅: TiCN
- 직경 범위: 1/4~1

## 대응 ISO 그룹
N

## 대상 피삭재
- Aluminum-wrought alloy
- Aluminum-cast, alloyed

## 특징
- TiCN coated
- 3 flute end mills
- Recommended cutting conditions for aluminum wrought alloy and aluminum cast alloyed materials
- Side cutting and slotting conditions provided

## 용도
- Aluminum-wrought alloy
- Aluminum-cast, alloyed

## 상세 정보 (원문 요약)
