## 상세 정보 (원문 요약)
- [page_0945] HSS-E NUT TAP용 선택 가이드 페이지로, Nut Tapping Machines에 사용하는 탭의 기본 사양을 보여준다. Hole type은 Max. 2.0xD Through Hole, Tool material은 HSS-E, Chamfer lead는 DIN2197 기준 Long, Flute type은 Straight Flute, Surface treatment는 Bright이다. 또한 M, MF, UNC, UNF, BSW, G(BSP), EG-M, EG-UNC, EG-UNF 등의 나사 규격별 대응 시리즈와, ISO P/M/K/N/S/H 재질군에 대한 절삭속도 권장 범위가 함께 제시되어 있다.
- [page_0409] NUT TAP TC803 SERIES는 ISO metric coarse threads DIN 13용 너트 탭이며, DIN 357 규격의 Hole type 2.0×D입니다. 기계에서 너트를 가공하는 용도이고, 완성된 공작물은 샹크 쪽으로만 배출할 수 있습니다. M4부터 M20까지의 규격이 있으며, 3날과 4날 구성이 포함됩니다. 적용 재질은 P/M/K/N/S/H 전 범위를 포괄하며, 각 재질군에 대해 ◯ 또는 공란으로 일반 적용성이 표시되어 있습니다.
