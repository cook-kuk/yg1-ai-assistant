- [page_1023] 4G MILL END MILLS의 CARBIDE, 2날 圆鼻 颈部加长 시리즈 페이지이다. 신형 코팅과 신형 공구 형상으로 절삭성과 내마모성이 우수하며, 프리하든강, 탄소강, 합금강 및 금형/부품 가공에 적합하다. 코너 R은 0.02mm부터 2.0mm까지, 유효길이와 전장도 다양한 규격으로 제공된다.
