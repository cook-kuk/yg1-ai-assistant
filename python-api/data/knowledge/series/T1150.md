# I-SP — T1150
**제품명**: I-SP LONG SHANK SPIRAL FLUTE TAPS for GENERAL PURPOSE

## 기본 사양
- 공구 타입: Tap
- 서브타입: Long Shank Spiral Flute
- 날 수: 3-4
- 공구 재질: HSS-E
- 코팅: R40 (Series), Homo (Surface)
- 생크: Long Shank
- 직경 범위: W1/8~W1"
- 깊이비: 2.5×D

## 대응 ISO 그룹
P

## 대상 피삭재
- Carbon Steels
- Alloy Steels

## 특징
- Suitable for tapping blind holes in carbon steels and alloy steels where chips are produced in a continuous coil state
- Long shank type
- Up to W5/16 : Male Center
- JIS II precision
- 55° thread angle
- 2.5P chamfer

## 용도
- Blind hole tapping
- General purpose tapping in carbon steels and alloy steels
