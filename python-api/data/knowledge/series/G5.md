## 선정 가이드
- 선정표로, 재질/나사 규격별로 탭 시리즈를 고르는 페이지이다. G5는 TiCN 코팅 HSS-E 스파이럴 플루트 탭으로 표시되며, 블라인드 홀 최대 2.5xD 용도이다.
- 선정표에서 G5는 Max. 2.5xD Blind Hole용 HSS-E 탭으로 표시되며, TiCN 코팅과 Spiral Flute, R50 형상으로 제시됨.
