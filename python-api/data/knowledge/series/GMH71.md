## 상세 정보 (원문 요약)
- [page_1242] PLAIN SHANK GMH71 SERIES는 V7 PLUS END MILLS의 6날 초경 엔드밀이다. 가변 피치(variable pitch) 형상으로 고속 및 트로코이달 밀링에서 진동을 줄이는 것이 특징이며, 스테인리스강, 연강, 주철 및 HRC40 이하의 저/중경도 소재에 우수한 성능을 보인다. 규격은 D6.0부터 D25.0까지이며, 절삭날 강화(Enforced Cutting Edge)가 적용되어 있다.
- [page_1247] V7 PLUS END MILLS 推荐加工参数页，适用于 GMH69、GMH71、GMH70 系列 6刃侧铣削。页面仅提供切削条件表，包含不同 ISO 材料组对应的 Vc、fz、RPM 和 FEED 参数，以及刀具直径 6.0~25.0 的推荐值。另有备注：当工件切深 L.O.C 低于 2D 时，AP 应按 L.O.C x 90% 应用。
