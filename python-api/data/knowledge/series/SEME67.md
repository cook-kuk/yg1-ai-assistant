# 4G MILLS — SEME67
**제품명**: 4날 다기능용 래디우스 / 4날 다기능용 롱넥 래디우스

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Radius / Long Neck Radius
- 날 수: 4
- 직경 범위: 3.0~20.0
- 깊이비: 2.0D

## 대응 ISO 그룹
P, H

## 대상 피삭재
- 합금강
- 탄소강
- 프리하든강
- 고경도강

## 특징
- 4날 다기능용
- 래디우스 및 롱넥 래디우스 형상

## 상세 정보 (원문 요약)
- [page_0285] 추천 절삭조건 4G MILLS의 SEME67 시리즈 절삭조건 페이지이다. 4날 다기능용 래디우스 / 4날 다기능용 롱넥 래디우스 제품으로, 인선 3.0~20.0mm에 대한 절삭조건이 제시되어 있다. P계열(합금강, 탄소강, 프리하든강)과 H계열(고경도강)용 Vc, Fz, RPM, Feed가 표로 제공되며, Ap는 2.0D, Ae는 재질에 따라 0.05D 또는 0.02D로 안내된다.
