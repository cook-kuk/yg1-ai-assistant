# YG-1 — E2749
**제품명**: 2날 엑스트라 롱 스퀘어

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Extra Long Square
- 날 수: 2
- 직경 범위: D2.0~D40.0

## 대응 ISO 그룹
P, N

## 대상 피삭재
- 탄소강
- 합금강
- 공구강
- 알루미늄
- 알루미늄 합금

## 특징
- 추천 절삭조건 페이지
- 2날 스퀘어 / 2날 롱 스퀘어 / 2날 엑스트라 롱 스퀘어 시리즈의 절삭조건 제공

## 상세 정보 (원문 요약)
- [page_0651] 추천 절삭조건 페이지로, 2날 스퀘어 / 2날 롱 스퀘어 / 2날 엑스트라 롱 스퀘어 E2401 / E2406 / E2749 시리즈의 절삭조건을 제시한다. ISO P(탄소강, 합금강, 공구강)와 ISO N(알루미늄, 알루미늄 합금) 가공에 대한 Vc, Fz, RPM, Feed 값이 인선경 D2.0~D40.0 범위로 정리되어 있다. Ap와 Ae는 도식으로 설명되며, 예시 조건으로 Ap 0.5D, Ae 1.0D가 표시되어 있다.
