# YG-1 — D2104
**제품명**: HSS Co8, STRAIGHT SHANK TWIST DRILLS

## 기본 사양
- 공구 타입: Drill
- 서브타입: Straight Shank Twist Drill
- 공구 재질: HSS Co8
- 코팅: Coloring (Gold color); optional TiN (D4104), TiCN (D7104), TiAlN (DQ104)
- 생크: Straight shank / cylindrical shank
- 직경 범위: D2.0~D12.0

## 대응 ISO 그룹
M, S, P, K, N

## 대상 피삭재
- Stainless steels
- Austenitic steels
- Titanium
- Inconel
- Difficult-to-cut materials

## 특징
- Long series straight shank twist drills
- Surface treatment: Coloring (Gold color)
- HSS-E (DL104) is available on request
- Optional coatings available on request: TiN, TiCN, TiAlN

## 용도
- Drilling deep holes in stainless steels
- Drilling difficult-to-cut materials such as titanium and inconel
- For drilling with drill bushes or in deep-seated locations

## 상세 정보 (원문 요약)
- [page_0149] D2104 시리즈는 HSS Co8 재질의 스트레이트 샹크 트위스트 드릴로, 표면처리는 골드 컬러의 Coloring입니다. 스테인리스강, 오스테나이트계 강, 티타늄, 인코넬 같은 난삭재의 심공 가공과 드릴 부시를 사용한 가공, 깊은 위치의 드릴 작업에 적합합니다. HSS-E(DL104)는 요청 시 가능하며, TiN(D4104), TiCN(D7104), TiAlN(DQ104) 코팅도 요청 시 제공됩니다.
- [page_0159] D2107, D1107, D2105, DL105, D1105, D1125, D2104, D1121, DL109 SERIES의 절삭조건 페이지로, HSS, HSS-E & HSS Co8 COBALT DRILLS에 대한 추천 절삭조건이 수록되어 있다. ISO P/M/K/N/S 재질별로 Vc, RPM, FEED가 정리되어 있으며, 지름 2.0~30.0mm 범위의 조건이 제시된다.
- [page_0944] 추천 절삭조건 페이지로, 코발트 드릴 D2101 / CD2101 / D2104 / D2107 / DL109 / D2105 / D2201 / DL205 시리즈의 절삭조건을 ISO P, M, K, N, S 재질별로 제시한다. 드릴직경 1.0, 2.0, 3.0, 4.0, 6.0, 8.0에 대해 Vc, RPM, Feed 범위를 제공하며, 저탄소강, 중탄소강, 고탄소강, 스테인레스강, 주철, 알루미늄, 티타늄 합금 등 적용 재질이 포함된다.
