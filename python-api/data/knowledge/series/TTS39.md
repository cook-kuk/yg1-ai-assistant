# SYNCHRO TAP — TTS39
**제품명**: SYNCHRO TAP - TTS39 SERIES / ISO Metric Coarse Threads DIN 13

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap / Synchro tap
- 공구 재질: HSS-PM
- 코팅: TIN
- 냉각: External
- 생크: DIN 371 / DIN 376
- 직경 범위: M3~M20
- 깊이비: 3.0xD

## 대응 ISO 그룹
P

## 대상 피삭재

## 특징
- Coated HSS-PM taps for high-speed tapping on rigid CNC machines or equivalent machines
- Up to 3 times faster in tapping compared to conventional taps
- For high-speed synchro tapping, synchro holder is recommended to increase tool life and thread quality
- High precision threads
- DIN 371 (M3-M10) and DIN 376 (M12-M20)
- Hole type: 3.0xD

## 용도
- ISO Metric Coarse Threads DIN 13
- Machine tapping on rigid CNC machines
- High-speed synchro tapping
