# YG-1 — TRG15
**제품명**: TRG15 シリーズ スパイラルポイントタップ (汎用)

## 기본 사양
- 공구 타입: Tap
- 서브타입: Spiral Point
- 생크: Square shank
- 직경 범위: M2~M20

## 대상 피삭재
- 延性材

## 특징
- 延性材に対しても高いパフォーマンスを発揮
- ねじの拡大防止、ねじ精度に関するトラブルを最小限に抑えるための設計

## 용도
- 汎用

## 상세 정보 (원문 요약)
- [page_0033] TRG15 シリーズは汎用のスパイラルポイントタップです。延性材に対しても高いパフォーマンスを発揮し、ねじの拡大防止とねじ精度トラブルの低減を狙った設計です。M2からM20までのねじサイズが掲載されており、等級はYH1.5〜YH5、溝数は2または3または4です。
