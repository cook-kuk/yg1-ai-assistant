# ALU-POWER — E5975
**제품명**: ALU-POWER (Inch) / 3 flute ball nose end mill

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Ball Nose
- 날 수: 3
- 코팅: TiCN Uncoated
- 헬릭스각: 40°
- 직경 범위: R3/64~R5/16

## 대응 ISO 그룹
N

## 대상 피삭재
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- 3 flute
- Ball nose cutting edge shape
- Long length neck
- TiCN Uncoated
- 3 flute plane end mill
- Recommended cutting conditions shown for small diameters and aluminum/copper alloys
- Long and extra long types should reduce feed by around 50%

## 용도
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Aluminum machining
- Copper alloy machining

## 선정 가이드
- Excellent (◎) for aluminum-wrought alloy and aluminum-cast, alloyed materials.

## 상세 정보 (원문 요약)
- [page_0917] ALU-POWER (Inch) 선택 가이드 페이지로, E5975는 3날 볼노즈 엔드밀이며 40° 헬릭스, LONG LENGTH NECK, TiCN Uncoated 사양이다. 알루미늄-단조합금과 알루미늄-주조합금에 대해 ◎로 표시되어 있다.
- [page_1623] E5975 시리즈 3날 - 플레인에 대한 추천 절삭조건이 제시되며, 3/32~5/8 범위의 직경에 대해 알루미늄-단조합금/알루미늄-주조합금(N, VDI 3323 21-25)과 구리 및 구리합금(청동/황동, N, VDI 3323 26-28) 조건이 포함된다. 장날 및 초장날 타입은 이송을 약 50% 줄이도록 안내한다.
