# COMBO TAP — TD834
**제품명**: COMBO TAP TiN / Unified coarse threads UNC / Machine taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 코팅: TiN
- 직경 범위: #4~1
- 깊이비: 3.0xD

## 특징
- For using multi-purpose and correct thread profiles
- Long tool life due to special tap geometry
- YG-1 company has a patent
- Hole type 3.0xD
- DIN 371(#4~3/8) and DIN 376(7/16~1)

## 용도
- Unified coarse thread tapping
- UNC tapping
- Machine tapping

## 상세 정보 (원문 요약)
- [page_0297] COMBO TAP의 TD834 SERIES는 TiN 코팅의 Unified coarse threads(UNC)용 머신 탭이다. 특수한 탭 형상으로 다양한 용도에 대응하고 정확한 나사 형상과 긴 공구 수명을 목표로 하며 YG-1 특허 기술이 적용되었다. Hole type은 3.0xD이고, #4부터 1까지의 규격이 수록되어 있다. DIN 371(#4~3/8) 및 DIN 376(7/16~1) 범위가 명시되어 있다.
