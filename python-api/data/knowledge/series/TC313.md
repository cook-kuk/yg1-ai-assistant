# YG TAP HARDENED — TC313
**제품명**: YG TAP HARDENED TC313 SERIES

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 날 수: 3 or 4
- 코팅: Bright
- 생크: DIN 371 / DIN 376
- 직경 범위: M2.5~M30
- 깊이비: 2.5×D

## 특징
- ISO metric coarse threads DIN 13
- Suitable for tapping blind holes due to special flute geometry and excellent chip evacuation
- DIN profile not ISO
- Hole type 2.5×D

## 용도
- Tapping blind holes
- Machine tapping

## 상세 정보 (원문 요약)
- [page_0346] YG TAP HARDENED TC313 SERIES는 ISO metric coarse threads DIN 13용 머신 탭 시리즈이다. 특수한 홈 형상과 우수한 칩 배출로 블라인드 홀 가공에 적합하며, DIN 371 / DIN 376 규격과 Hole type 2.5×D가 명시되어 있다. M2.5부터 M30까지 다양한 규격이 제공된다.
