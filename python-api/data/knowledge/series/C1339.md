# None — C1339
**제품명**: 세날타입 C1339 시리즈

## 기본 사양
- 공구 타입: Countersink
- 서브타입: 3-flute type
- 날 수: 3
- 직경 범위: Ø5.0~Ø50.0

## 대응 ISO 그룹
P, M, K, N

## 대상 피삭재
- 탄소강
- 스테인레스 강
- 회주철
- 구상흑연주철
- 가단주철
- 알루미늄 단조합금
- 알루미늄 주조합금
- 구리 및 구리합금

## 상세 정보 (원문 요약)
- [page_1353] 추천 절삭조건 카운터싱크 페이지로, 세날타입 C1339 시리즈의 절삭조건이 ISO 재질군별로 정리되어 있다. 탄소강, 스테인레스 강, 회주철/구상흑연주철/가단주철, 알루미늄 단조합금/주조합금, 구리 및 구리합금에 대해 Vc와 Ø5.0~Ø50.0별 이송값이 제시되어 있다. 이 페이지는 제품 소개가 아니라 추천 절삭조건 표이며, 시리즈명과 적용 재질 정보만 제공한다.
