# X-POWER — GM639
**제품명**: CARBIDE, 4 FLUTE STUB LENGTH CORNER RADIUS

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Stub Length Corner Radius
- 날 수: 4
- 공구 재질: CARBIDE
- 코팅: Y Coating
- 헬릭스각: 30°
- 생크: PLAIN
- 직경 범위: 1/16~1/2
- 깊이비: 2.5D

## 상세 정보 (원문 요약)
- [page_1361] X-POWER PRO END MILLS의 CARBIDE, 4 FLUTE STUB LENGTH CORNER RADIUS 시리즈 페이지이다. 30° 헬릭스, PLAIN 샹크, Y Coating 사양이며, tool steel, alloy steel, mold steel 및 기타 고경도 소재 가공용으로 설계되었다. 인치 단위로 여러 코너 반경과 직경 조합의 EDP No.가 제시되어 있고, 추천 소재 표에서 P/M/K/H 계열 소재에 대한 적합도가 표시된다.
- [page_1398] X-POWER PRO END MILLS의 추천 절삭조건 페이지로, GM639, GM649, GM212 시리즈 4날 코너 라디우스 엔드밀의 사이드 커팅 조건을 제시한다. ISO P와 H 소재군에 대해 비합금강, 저합금강, 고합금강/공구강, 경화강, 칠드 주철, 경화주철의 절삭조건이 정리되어 있으며, 절입은 주로 2.5D, Ae는 0.05D 또는 0.02D로 안내된다. 지름은 1/4, 5/16, 3/8, 1/2에 대한 SFM, IPT, RPM, IPM 값이 표로 제공된다.
