# YG-1 — D1123
**제품명**: 스트레이트 생크 드릴 D1101 / CD1101 / D1123 / D1102 / D1121 / D1107 / D1105 / D1106 / D1100 시리즈

## 기본 사양
- 공구 타입: Drill
- 서브타입: Straight Shank
- 공구 재질: HSS
- 생크: Straight Shank
- 직경 범위: D0.5~D8.0

## 대응 ISO 그룹
P, M, K, N, S

## 상세 정보 (원문 요약)
- [page_1001] 추천 절삭조건 - 하이스 드릴 페이지로, 스트레이트 생크 드릴 D1101 / CD1101 / D1123 / D1102 / D1121 / D1107 / D1105 / D1106 / D1100 시리즈의 절삭조건이 수록되어 있다. ISO P/M/K/N/S 재질별로 Vc, RPM, Feed가 드릴 직경 0.5~8.0에 대해 제시되어 있으며, 저탄소강, 일반강, 스테인레스강, 주철, 알루미늄, 티타늄 합금 등에 대한 추천 조건이 포함된다.
