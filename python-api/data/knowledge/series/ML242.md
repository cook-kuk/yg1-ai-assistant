## 상세 정보 (원문 요약)
- [page_1496] HSS-E, DOVETAIL CUTTERS TYPE “B”, “D”, “F”의 THREAD SHANK 시리즈로 ML242 시리즈가 수록되어 있다. 표에서는 ML24202001 한 규격이 표시되며, 60° 타입과 20.0mm 직경, 8mm 날폭, 12mm 샹크, 63mm 전장, 6날 사양이다. ‘▲’ 표시는 재고 소진 시까지만 공급됨을 의미한다.
- [page_1516] 추천 절삭조건 페이지로, ML032, ML132, ML042, ML142, ML232, ML242 시리즈의 DOVETAIL CUTTERS TYPE 'B', 'D', 'F' / 燕尾槽铣刀 B型, D型, F型에 대한 절삭조건이 수록되어 있다. 본 페이지는 Vc, fz, RPM, FEED의 수치표만 제공하며, 재질별로 P계열(Non-alloy steel, Low alloy steel, High alloyed steel, and tool steel)과 N계열(Aluminum-wrought alloy, Aluminum-cast, alloyed) 조건이 제시된다.
