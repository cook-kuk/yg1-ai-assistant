# YG-1 — E5E96
**제품명**: 초경 코너라운딩 E5E96 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Corner Rounding
- 공구 재질: CARBIDE

## 대응 ISO 그룹
P

## 대상 피삭재
- 탄소강
- 합금강
- 공구강

## 상세 정보 (원문 요약)
- [page_1347] 추천 절삭조건 페이지로, 초경 코너라운딩 E5E96 시리즈의 절삭조건이 ISO P 재질(탄소강, 합금강, 공구강)과 경도 구간별로 제시되어 있다. 인선(mm) 0.5, 1.5, 1.9, 3.9에 대해 Vc, Fz, RPM, Feed 값이 표로 제공된다. 코팅제품인 EHE96 시리즈는 Feed를 20% 상향하여 작업하는 것을 추천한다.
