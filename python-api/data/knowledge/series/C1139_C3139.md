# YG-1 — C1139/C3139
**제품명**: HSS & HSS Co8, THREE FLUTE COUNTERSINKS (90°)

## 기본 사양
- 공구 타입: Countersink
- 서브타입: Three Flute
- 날 수: 3
- 공구 재질: HSS / HSS Co8
- 코팅: Bright (TIN & TiCN available on request)
- 선단각: 90°
- 생크: Straight shank
- 직경 범위: D1 4.3~31.0

## 특징
- Self-centering (3 flutes)
- Designed for 90° capscrews countersinking
- Hand using
- Longitudinal chamfers and contouring
- Works without vibrations
- Besonders geeignet zum 90° Ansenken für Senkkopfschrauben

## 용도
- 90° capscrews countersinking
- Deburring
- Chamfering
- Contouring
- Manual use

## 상세 정보 (원문 요약)
- [page_0225] DIN 335 C 규격의 HSS/HSS Co8 3날 센터링커(90°) 시리즈로, 90° 캡스크류용 카운터싱킹에 적합하며 자기중심형이다. 수동 사용이 가능하고 진동 없이 작업할 수 있으며, TIN 및 TiCN 코팅은 요청 시 제공된다. 절삭날 직경은 4.3~31.0mm 범위이고 공차는 커터 직경 ±0.05mm, 샹크 직경 h9, 포인트 각도 +0/-1°이다.
