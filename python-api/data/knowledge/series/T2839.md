## 선정 가이드
- Use T2839 for max. 2.5xD blind holes. It is the UNC/F version of the spiral flute combo tap with HSS-E material, 2.5P chamfer lead, R40 spiral flute angle, JIS Type I, and bright surface treatment.
- Hole type: 2.5×D. Tap drill sizes are referenced on p.B233-B238. Material recommendation chart uses ◎/○ style ratings; this page shows recommended materials across P/M/K/N/S/H groups.

## 상세 정보 (원문 요약)
- [page_0615] THREADING TOOLS HSS-E COMBO TAPS (For Multi Purpose Tapping 多用途丝锥) 페이지의 선택 가이드이다. T2839는 Max. 2.5xD 블라인드 홀용이며 HSS-E 재질, Spiral Flute, 2.5P 챔퍼, R40 나선각, JIS Type I, Bright 표면처리로 표시된다. 적용 재질 표에는 P/M/K/N/S 계열의 다양한 재질에 대한 적합도가 ◎/○로 제시되어 있다.
- [page_0621] T2839 SERIES COMBO TAPS는 UNC/F I-SP SPIRAL FLUTE TAPS for Multi Purpose 제품이다. 특수한 tap geometry로 다목적 가공과 올바른 나사 형상을 구현하며, 공구 수명을 향상시킨다. 스테인리스 가공에 적합하고, 기본 코팅은 Bright이며 TiCN 또는 TiAlN 코팅은 요청 시 가능하다. Hole type은 2.5×D이고, #2-56UNC부터 7/16-20UNF까지 다양한 규격이 수록되어 있다.
