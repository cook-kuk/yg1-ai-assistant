## 선정 가이드
- ◎ : Excellent, ○ : Good. This series is selected for M thread form, carbide material, helix flute type, and R15 helix angle; the table indicates material suitability ratings across ISO groups.
- Select TD for M thread form, blind/through holes, and TiAlN-coated solid carbide thread milling without coolant hole.

## 상세 정보 (원문 요약)
- [page_0529] THREADING TOOLS 선택 가이드 페이지로, TD 시리즈는 without Coolant Hole, CARBIDE, Helix, R15 조건의 M 나사 가공용으로 제시된다. 표에서는 각 재질군에 대한 적합도(◎/○)가 함께 표시되며, P/M/K/N/S/H ISO 그룹 전반의 적용 재질이 안내된다.
- [page_0571] TD는 SOLID CARBIDE THREAD MILL 시리즈로, M 나사 형식의 블라인드/관통 홀 가공에 사용된다. 냉각홀 없는 타입이며 TiAlN 코팅이 적용된다. 다양한 재질 범위에서 같은 스레드밀로 여러 홀 크기를 가공할 수 있도록 안내되어 있다.
