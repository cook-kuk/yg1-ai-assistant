# YG TAP GENERAL — TC124
**제품명**: TC124 SERIES - Unified fine threads

## 기본 사양
- 공구 타입: Machine tap
- 서브타입: Unified fine threads / UNF
- 날 수: 3-4
- 생크: DIN 371 / DIN 374
- 직경 범위: #5~7/8
- 깊이비: 2.5×D

## 특징
- Suitable for tapping blind holes due to special flute geometry
- Excellent chip evacuation
- UNF unified fine threads
- Machine taps

## 용도
- Tapping blind holes

## 상세 정보 (원문 요약)
- [page_0383] TC124 SERIES는 UNF(Unified fine threads)용 머신 탭이며, 특수한 홈 형상과 우수한 칩 배출로 블라인드 홀 가공에 적합합니다. Hole type은 2.5×D이고, DIN 371 및 DIN 374 규격이 적용됩니다. 표에는 #5부터 7/8까지의 치수, TPI, 전체 길이, 넥 길이, 샹크 직경, 스퀘어 크기, 플루트 수, 드릴 직경이 제시되어 있습니다.
