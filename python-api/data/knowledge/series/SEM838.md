# X-POWER S — SEM838
**제품명**: 2날 롱 생크 볼 SEM838 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Ball / Long Shank
- 날 수: 2
- 생크: Long Shank

## 대응 ISO 그룹
P, H

## 대상 피삭재
- 탄소강
- 합금강
- 고경도강

## 특징
- 2날 롱 생크 볼 엔드밀 시리즈
- 추천 절삭조건 페이지로, 재질별 절삭속도와 이송 조건을 제시함

## 상세 정보 (원문 요약)
- [page_0430] 2날 롱 생크 볼 SEM838 시리즈의 추천 절삭조건 페이지이다. 탄소강(P, ~30), 합금강(P, 30~40), 고경도강(H, 45~65)에 대해 인선별 Vc, Fz, RPM, Feed 조건이 표로 제시되어 있다. Ap와 Ae는 일부 조건에서만 제한적으로 표시되며, 본 페이지는 절삭조건 안내가 मुख्य 내용이다.
