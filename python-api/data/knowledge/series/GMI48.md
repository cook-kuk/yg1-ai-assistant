## 상세 정보 (원문 요약)
- [page_0517] 추천 절삭조건 V7 PLUS / 4날 칩브레이커 GMI48 시리즈의 슬로팅 절삭조건 페이지이다. 인선(mm) 6.0, 8.0, 10.0, 12.0, 16.0에 대해 각 소재별 Vc, Fz, RPM, Feed가 제시되어 있으며, Ap와 Ae는 대부분 1.0D로 표시된다. 소재는 탄소강, 합금강, 공구강, 주철, 스테인레스강 300/400/PH, 티타늄, 내열합금이 포함된다.
- [page_0518] 추천 절삭조건 V7 PLUS 페이지로, 4날 칩브레이커 GMI48 시리즈의 사이드커팅 절삭조건을 제시한다. ISO P/K/M/S 재질별로 탄소강, 합금강, 공구강, 주철, 스테인레스강 300/400/PH, 티타늄, 내열합금에 대한 Vc, Fz, RPM, Feed가 표기되어 있다. 절삭 조건은 인선 6.0~16.0mm 기준이며, Ap/Ae는 재질군별로 1.5D 또는 1.0D, Ae 0.5D/0.35D/0.25D로 안내된다.
