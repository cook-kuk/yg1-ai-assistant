# M42 HSS — CE7412
**제품명**: 4날 스퀘어 / 롱 스퀘어 CE7412 / CE7659 시리즈

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Square / Long Square
- 날 수: 4
- 공구 재질: M42 HSS
- 직경 범위: D2.0~D40.0

## 대응 ISO 그룹
P, N

## 대상 피삭재
- 탄소강
- 합금강
- 공구강
- 알루미늄
- 알루미늄 합금

## 상세 정보 (원문 요약)
- [page_0734] 추천 절삭조건 M42 HSS의 4날 스퀘어 / 롱 스퀘어 CE7412 / CE7659 시리즈 절삭조건 페이지이다. ISO P와 N 재질에 대해 직경별 Vc, Fz, RPM, Feed가 표로 제시되어 있으며, Ap와 Ae는 특정 조건에서 1.5D, 0.1D로 안내된다. 하단에는 Ap는 축방향 절삭깊이, Ae는 반경방향 절삭깊이의 정의가 설명되어 있다.
