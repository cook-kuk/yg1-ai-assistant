# YG-1 — Spade Drill
**제품명**: Spade Drill (하이스)

## 기본 사양
- 공구 타입: Drill
- 서브타입: Spade Drill
- 공구 재질: HSS
- 직경 범위: Ø9.5~114.0

## 대응 ISO 그룹
K, N, S

## 대상 피삭재
- K: 주철/회주철
- N: 알루미늄
- S: 내열합금강

## 특징
- 추천 절삭조건 표
- HSS GRADE별 절삭조건 제공
- 코팅별 Vc(TiN, TiCN, TiAlN)와 Feed 값 제시
- 드릴 직경 구간별 조건 제공

## 용도
- 주철/회주철
- 알루미늄
- 내열합금강

## 상세 정보 (원문 요약)
- [page_1064] 추천 절삭조건 페이지로, Spade Drill(하이스)의 ISO별 절삭조건을 드릴 직경 구간별로 제시한다. K(주철/회주철), N(알루미늄), S(내열합금강) 소재에 대해 TiN, TiCN, TiAlN 코팅별 Vc와 Feed 값이 표로 정리되어 있다. 하단에는 HSS=HSS M4, SH=Super HSS T15, PH=Premium HSS M48의 약어 설명과 RPM 계산식, 그리고 이상적인 조건이므로 장비/작업 여건에 따라 값을 줄여 사용하라는 주의사항이 있다.
