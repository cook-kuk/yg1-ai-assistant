## 상세 정보 (원문 요약)
- [page_0259] ER COLLET CHUCK 시리즈로, JIS B6339 / MAS 403-BT 규격의 BT 테이퍼 ER 콜렛 척입니다. BT30, BT40, BT50 모델이 있으며 ER16/20/25/32/40 콜렛과 조합되는 다양한 길이와 클램핑 범위를 제공합니다. ER collet은 D38-D39, ER nut과 spanner는 D43 페이지를 참조하도록 안내되어 있습니다.
- [page_0264] ER는 CBT(BT DUAL CONTACT) 규격의 ER COLLET CHUCK이다. 표준은 JIS B6339/MAS 403이며, 콜렛·너트·스패너는 D40-D41 페이지를 참조한다. 이 페이지에는 CBT40-ER16-75, CBT40-ER16-100, CBT40-ER16-120, CBT40-ER25-100, CBT40-ER32-100 모델의 클램핑 범위와 치수가 수록되어 있다.
- [page_0269] ER COLLET CHUCK 제품 페이지로, DIN 228-MTA 규격의 툴링 시스템이다. MTA2, MTA3, MTA4, MTA5에 대응하는 ER20/25/32/40 모델과 클램핑 범위, 치수 L/D/H, 중량이 표로 정리되어 있다.
