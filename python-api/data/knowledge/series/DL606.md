## 상세 정보 (원문 요약)
- [page_0363] DL603 SERIES / DL606 SERIES는 HSS Co5 재질의 AIRCRAFT EXTENSION DRILL 135° SPLIT POINT 제품이다. Right hand spiral, 30° helix, 135° split point 사양이며, 대부분의 소재에서 칩 배출이 향상되고 특히 deep drilling applications에 적합하다. ISO Application Table 기준으로 P/M/K/N/S/H 계열 소재 전반에 적용 범위가 제시되어 있다.
- [page_0368] AIRCRAFT DRILLS RECOMMENDED CUTTING CONDITIONS 페이지로, DL606 시리즈의 추천 절삭조건을 제시한다. 본 페이지는 DL601, DL602, DL603, D1631, D1632, D1633 및 DL604, DL605, DL606, D1634, D1635, D1636 시리즈의 절삭조건 표를 포함하지만, 제품의 상세 사양은 없고 헤더 수준의 정보만 제공한다.
