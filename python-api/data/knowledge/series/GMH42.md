# SUS-CUT — GMH42
**제품명**: 4날 롱넥 스퀘어

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Long Neck Square
- 날 수: 4

## 대응 ISO 그룹
P, M

## 상세 정보 (원문 요약)
- [page_0461] 추천 절삭조건 SUS-CUT의 4날 스퀘어 / 4날 롱넥 스퀘어 EMD82 / GMH42 시리즈 절삭조건 페이지이다. 본문은 합금강(P)과 스테인레스강 300/400(M)에 대한 Vc, Fz, RPM, Feed, Ap, Ae의 추천 절삭조건 표로 구성되어 있다. 제품 헤더상 EMD82와 GMH42 시리즈가 언급되지만, 상세 제품 설명은 없고 절삭조건만 제공된다.
