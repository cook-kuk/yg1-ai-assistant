# None — DY301
**제품명**: 하이스 센터 드릴

## 기본 사양
- 공구 타입: Drill
- 서브타입: Center Drill
- 공구 재질: HSS
- 직경 범위: D0.5~D10.0

## 대응 ISO 그룹
P, M, K

## 대상 피삭재
- 저탄소강
- 중탄소강
- 고탄소강
- 저합금강
- 스테인레스 강
- 회주철
- 구상흑연주철
- 가단 주철

## 상세 정보 (원문 요약)
- [page_1019] 추천 절삭조건 센터 & NC 드릴 페이지로, 하이스 센터 드릴 DV301 / DY301 / DV304 / DV305 시리즈의 절삭조건이 수록되어 있다. 본문은 재질별 Vc, RPM, Feed 표이며, 저탄소강, 중탄소강, 고탄소강, 저합금강, 스테인레스 강, 회주철, 구상흑연주철, 가단 주철에 대한 추천 조건을 제시한다. 상단 제목 외의 제품 특징, 코팅, 형상 상세는 없고, 직경 범위는 0.5~10.0mm로 보인다.
