- [page_1721] HSS Co8, 2날 롱 스트레이트/롱 길이의 볼노즈 엔드밀 페이지로, DIN 1889, 30° 헬릭스, R ±0.02 정밀도를 가진다. 기본 코팅은 UNCOATED와 TiAlN이며, TiN과 TiCN 코팅도 요청 시 가능하다. DIN 1835B 샹크를 사용하고, 밀 직경 공차는 0 ~ -0.03 mm, 샹크 공차는 h6이다.

- [page_1740] HSS Co8, 4날 숏 길이 엔드밀 시리즈로 DIN 844 규격이며, 약 30° 헬릭스와 DIN 1835B 샹크를 사용한다. Ø3mm 이하에서는 센터컷 타입이며, 기본은 UNCOATED와 TiAlN 사양이 제시되고 TiN/TiCN 코팅은 요청 시 가능하다. 지름은 D2.0~D20.0까지 있으며, 공차는 밀 직경 0~+0.04mm, 샹크 직경 h6이다. ISO P/M/K/N/S/H 재질군에 대해 추천 정보가 함께 수록되어 있다.
