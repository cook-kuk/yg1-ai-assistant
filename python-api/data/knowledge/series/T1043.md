- [page_0729] T1043 SERIES는 SPIRAL POINT TAPS for STAINLESS STEELS / 不锈钢用先端丝锥 시리즈이다. 스테인리스강, 크롬강, 크롬몰리브덴강 및 가공경화 경향이 큰 연성 재료의 통공 탭 가공에 적합하며, Hole type은 3.0×D이다. up to M8은 Male Center 사양이며, 규격은 M14 x 1.5부터 M30 x 1.5까지 수록되어 있다.
