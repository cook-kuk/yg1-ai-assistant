# SYNCHRO TAP — TTS43-IC
**제품명**: SYNCHRO TAP TTS43-IC

## 기본 사양
- 공구 타입: Tap
- 서브타입: Synchronous tap

## 대응 ISO 그룹
P, M, K, N

## 상세 정보 (원문 요약)
- [page_0665] SYNCHRO TAP 절삭조건 표로, TTS41-IC와 TTS43-IC의 적용 재질별 VDI 3323 재료군이 정리되어 있다. P, M, K, N ISO 그룹에 대해 비합금강, 저합금강, 스테인리스강, 회주철, 구상흑연주철, 가단주철, 알루미늄 합금, 구리 및 구리합금 등의 경도 범위와 대응 절삭조건 범위가 제시되어 있다.
