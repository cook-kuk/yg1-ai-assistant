# YG TAP FORMING — TE723
**제품명**: ISO metric coarse threads DIN 13 / Cold forming taps / Gewindeformer

## 기본 사양
- 공구 타입: Tap
- 서브타입: Cold forming tap
- 생크: DIN 371(M2~M10) and DIN 376(M11~M20)
- 직경 범위: M2~M20
- 깊이비: 3.0×D

## 대응 ISO 그룹
P, M, K, N, S, H

- Copper and Copper Alloys (Bronze / Brass)

- Hardened Cast Iron

## 특징
- Suitable for threading soft materials with at least 8-10% elongation
- The pre-drilling holes are bigger than normal sized holes
- Hole type 3.0×D
- DIN profile not ISO
- ISO metric coarse threads DIN 13

## 용도
- Thread forming for ISO metric coarse threads DIN 13
- Cold forming tapping for soft materials

## 상세 정보 (원문 요약)

- [page_0372] TE723 시리즈는 ISO metric coarse threads DIN 13용 cold forming tap(gewindeformer)이다. 연신율 8~10% 이상의 연질 소재에 적합하며, 일반 탭보다 큰 선가공 구멍을 사용한다. Hole type은 3.0×D이고, DIN 371(M2~M10) 및 DIN 376(M11~M20) 규격을 따른다.
