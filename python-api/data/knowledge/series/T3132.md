## 상세 정보 (원문 요약)
- [page_0634] Max. 2.5xD Blind Hole용 HSS-E Spiral Flute Tap 선택 가이드 페이지이다. T3132는 JIS Type, R40, TiN 코팅으로 표시되며, P/M/K/N/S/H 재질군에 대해 적용 범위가 표기되어 있다. 이 페이지는 각 재질군별로 탭 번호에 따라 추천 시리즈를 매칭해 주는 선택표이다.
- [page_0665] I-SP 螺旋槽丝锥 T3132 SERIES。适用于碳钢和合金钢的盲孔攻丝，具有高效、寿命长、能够快速攻丝，孔类型为 2.5×D。规格范围从 M3 到 M20，技术尺寸表列出了不同牙距对应的型号、全长、螺纹长、柄径、方块尺寸和槽数。适用性表显示其主要推荐用于 ISO P、M、K、N、S、H 各类材料，其中对非合金钢和低合金钢为优秀，其余多为良好。
