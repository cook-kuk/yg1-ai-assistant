## 상세 정보 (원문 요약)
- [page_0284] CARBIDE DRILLS (D5401 SERIES) / 硬质合金钻头 (REGULAR / 常规) 是一页尺寸表，列出了 Bright 直柄硬质合金钻头的型号、刃径、槽长和全长。页面注明 D1 = D2，并给出推荐刀柄为 SHRINK FIT HOLDER、HYDRAULIC CHUCK、ER COLLET CHUCK。应用范围包括一般钢、铸钢、铸铁、冷硬铸铁、可锻铸铁、非铁重金属、非铁轻金属及耐磨塑料；下方应用表覆盖 ISO P/M/K/N/S/H 材料分类。
- [page_0285] D5401 SERIES / 常规 是一款硬质合金钻头，D1=D2，标注为 Bright，几何参数包含 30°、h6、h8、118°。页面给出了推荐刀柄：SHRINK FIT HOLDER（D47-72）、HYDRAULIC CHUCK（D15-46）、ER COLLET CHUCK（D73-115）。规格表列出刃径 D7.2 到 D13.0 的多种型号及对应槽长、全长。
- [page_0286] D5401 SERIES CARBIDE DRILLS 硬质合金钻头의 추천 절삭조건 페이지입니다. ISO P/M/K/N/S 재질별로 Vc, RPM, FEED가 드릴 직경 1.0~13.0mm에 대해 제시되어 있습니다. 본 페이지는 절삭조건 표 중심이며, 제품 특징이나 코팅 정보는 포함되어 있지 않습니다.
- [page_0890] 추천 절삭조건 페이지로, H4401 / D5401 / D5491 / D5405 / D5407 시리즈의 절삭조건을 ISO 재질별로 제시한다. 일반강, 스테인레스강, 회주철, 알루미늄 합금, 티타늄 합금에 대해 드릴 직경 1.0~5.0mm 기준의 Vc, RPM, Feed가 표기되어 있다. 상단에는 ●가 BEST, ○가 GOOD임을 안내한다.
- [page_0891] 추천 절삭조건 페이지로, 일반 초경 드릴/비철용 초경 드릴 H4401 / D5401 / D5491 / D5405 / D5407 시리즈의 절삭조건을 제시한다. ISO P, M, K, N, S 재질별로 권장 Vc, RPM, Feed가 드릴 직경 6.0, 8.0, 10.0, 12.0, 13.0에 대해 표기되어 있다. 상단에는 ● BEST, ○ GOOD 기준이 있으며, 제품시리즈 마지막 페이지로 표시되어 있다.
