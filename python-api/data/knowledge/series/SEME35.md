## 상세 정보 (원문 요약)
- [page_1052] PLAIN SHANK SEME35 SERIES는 4G MILL END MILLS의 초경 2날 엔드밀 시리즈이다. 신코팅과 공구 형상 적용으로 절삭성과 내마모성이 우수하며, 금형·다이용 예경화강(HRC55 이하) 가공에 뛰어난 성능을 보인다. 날끝의 예리한 형상 설계로 절삭성이 향상되며, 평행 생크(Plain Shank) 타입이다.
- [page_1053] CARBIDE, 2刃 엔드밀 시리즈로, 신코팅과 공구 형상을 적용하여 절삭성능과 내마모성을 향상시킨 제품이다. 모듈/금형용의 예경화강(HRc55 이하) 가공에 우수하며, 날끝의 예리한 형상으로 절삭능력을 높였다. 기술 치수는 D17.0~D25.0 범위이며, 플랫 샹크/플레인 샹크용 추천 홀더와 ISO P/M/K/N/S/H 적용성이 함께 제시되어 있다.
- [page_1054] PLAIN SHANK SEME35 SERIES는 4G MILL END MILLS의 초경 2날 엔드밀 시리즈로, 4mm 샹크를 적용한 제품이다. 신코팅과 공구 형상으로 절삭성과 내마모성을 높였으며, 금형·다이용 예경화강(HRC55 이하) 가공에 우수하다. 날끝의 날카로운 형상으로 작업 중 절삭 성능을 향상시킨다.
- [page_1123] SEME35 SERIES의 2날 슬롯 밀링 추천 절삭조건 페이지이다. ISO P/M/K/H 재질별로 Vc, fz, RPM, FEED 값이 표로 정리되어 있으며, 절삭폭 Ae는 1.0D로 제시된다. 절삭깊이 Ap는 일반적으로 0.5D이며, Ø3 이하와 Ø1 이하에서는 더 얕은 조건이 별도로 안내된다.
