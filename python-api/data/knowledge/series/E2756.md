# YG-1 — E2756
**제품명**: 3날 AL용 라핑 / 롱 라핑

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Roughing / Long Roughing
- 날 수: 3
- 공구 재질: M42 HSS
- 직경 범위: D6.0~D50.0

## 대응 ISO 그룹
N

## 대상 피삭재
- 알루미늄
- 알루미늄 합금

## 특징
- AL용
- 라핑
- 롱 라핑용

## 용도
- 알루미늄 가공
- 알루미늄 합금 가공

## 상세 정보 (원문 요약)
