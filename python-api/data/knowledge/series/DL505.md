- [page_0397] STRAIGHT SHANK DRILLS / DL510, DL508, DL509, DL505, DL504, DL608 SERIES. HSS-E, DH100 WORM PATTERN DRILLS로 표시된 추천 절삭조건 페이지이며, 본문은 ISO P/K 재질별 절삭조건 표로 구성되어 있습니다. 직선 생크 드릴의 권장 Vc, RPM, FEED가 지름 D2.0~D30.0 범위로 제시됩니다.
