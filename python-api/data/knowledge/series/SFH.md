**제품명**: SHRINK FIT HOLDER

- 공구 타입: Holder
- 서브타입: Shrink Fit Holder
- 냉각: External
- 생크: BT taper
- 직경 범위: d1 4~20 mm

## 특징
- JIS B6339 / MAS 403-BT
- Without balancing screw
- Resealable Coolant Channel type is available upon request
- DIN 69893 / ISO 12164-1-HSK FORM A

## 상세 정보 (원문 요약)
- [page_0259] SHRINK FIT HOLDER SFH 시리즈로, JIS B6339 / MAS 403-BT 규격의 BT 테이퍼 홀더입니다. 표에는 BT40 및 BT50 모델의 치수, EDP 번호, 클램핑 직경(d1), 길이(L), 나사 규격(G), 중량이 수록되어 있으며, 밸런싱 스크류는 없습니다. 별도로 SHRINK FIT HOLDER (COOLANT CHANNEL) 모델 BT40AD/B-SFH20C-90이 있으며, 요청 시 resealable coolant channel type을 제공합니다.
- [page_0267] SFH SHRINK FIT HOLDER는 DIN 69893 / ISO 12164-1 HSK FORM A 규격의 shrink fit holder이며, balancing screw가 없다. 일반형과 coolant channel 타입이 함께 수록되어 있다.
- [page_0267] SFH SHRINK FIT HOLDER (COOLANT CHANNEL)는 DIN 69893 / ISO 12164-1 HSK FORM A 규격의 냉각 채널형 shrink fit holder이다. HSK63A 기준으로 여러 d1 규격(6, 8, 10)이 제시되어 있다.
