# YG-1 — DH302
**제품명**: 초경 센터 드릴 D5302 / DH302 시리즈

## 기본 사양
- 공구 타입: Drill
- 서브타입: Center Drill
- 공구 재질: CARBIDE
- 직경 범위: 1.0~6.0

## 대응 ISO 그룹
P, M, K

## 대상 피삭재
- 저탄소강
- 일반강
- 중탄소강
- 고탄소강
- 저합금강
- 스테인레스 강
- 회주철
- 구상흑연주철
- 가단 주철

## 상세 정보 (원문 요약)
- [page_1017] 추천 절삭조건 페이지로, 초경 센터 드릴 D5302 / DH302 시리즈의 절삭조건을 ISO P, M, K 재질별로 제시한다. 드릴 직경 1.0~6.0에 대해 Vc, RPM, Feed 값이 표로 정리되어 있으며, 저탄소강, 일반강, 스테인레스 강, 회주철, 구상흑연주철, 가단 주철 등에 대한 권장 조건이 포함된다. 상단 헤더 외에는 제품 특징이나 구조 설명은 없고, 본 페이지의 핵심은 가공 조건 데이터이다.
