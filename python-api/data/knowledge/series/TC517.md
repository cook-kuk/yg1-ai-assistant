# YG-1 — TC517
**제품명**: TC517 SERIES ISO metric coarse threads DIN 13 / Machine taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 날 수: 3-4
- 코팅: Bright
- 생크: DIN 371 / DIN 376
- 직경 범위: M2~M30
- 깊이비: 2.5×D

## 특징
- Suitable for tapping blind holes due to special flute geometry
- Excellent chip evacuation
- DIN 371 for M2~M10 and DIN 376 for M11~M30
- DIN profile not ISO

## 용도

- ISO metric coarse threads DIN 13
