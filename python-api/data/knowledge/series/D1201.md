## 상세 정보 (원문 요약)
- [page_0355] D1201 시리즈는 HSS 모스 테이퍼 샹크 드릴로, 표면처리는 Steam Homo(흑색 산화피막)이다. 적용 재질은 강, 합금/비합금 주강, 회주철, 가단주철이며, 형상 정보로 N 30°, 118° 포인트가 표시되어 있다. 규격은 D13.0~D29.5 범위로 제공되며, 모스 테이퍼 번호 1, 2, 3이 포함된다.
- [page_0356] HSS, 莫氏锥柄麻花钻头（REGULAR / 常规）D1201 系列，表面处理为氧化发黑处理（Steam Homo / Black Oxide Finish）。适用于钢、铸钢合金和非合金钢、灰铸铁、可锻铸铁等材料加工。页面下方给出了 D30.0~D60.0 的尺寸表，以及按 ISO P/M/K/N/S/H 的应用推荐等级。
- [page_0365] D1201 시리즈는 HSS & HSSCo8 모스 테이퍼 샹크 드릴이다. 이 페이지는 추천 절삭조건 표로, P/M/K/N/S 재질군별 절삭속도(Vc), RPM, FEED 값을 제시한다. 제품의 상세 형상 정보는 없고, 시리즈명과 재질 적용 범위만 확인된다.
- [page_0047] D1201 시리즈는 HSS 모스 테이퍼 샹크 드릴이며, 표면처리는 Steam Homo(Black Oxide Finish)입니다. 강, 주철강(합금/비합금), 회주철, 가단주철의 드릴링에 적용됩니다. 규격은 D13.0부터 D43.0까지이며, 절삭조건 추천표에서는 다양한 ISO 재질군에 대해 폭넓게 적용 가능함을 보여줍니다.
- [page_0048] D1201 시리즈는 HSS 모스 테이퍼 샹크 드릴의 REGULAR 타입이다. 표면처리는 Steam Homo(Black Oxide Finish)이며, 강, 주강(합금/비합금), 회주철, 연성주철 가공에 적용된다. 규격 표에는 D44.0부터 D60.0까지의 드릴 직경과 플루트 길이, 전체 길이, 모스 테이퍼 번호가 수록되어 있다.
- [page_1004] 하이스 드릴의 테이퍼 생크 드릴, 노스 드릴 D1201 / D1206 / D1203 / D1202 / D1204 / D1208 / D1207 / D1209 / D1210 / D1205 / D1953 시리즈에 대한 추천 절삭조건 페이지이다. ISO P/M/K/N/S 재질별로 BEST(●)와 GOOD(○) 적용 추천이 표시되며, 드릴 직경 30.0, 40.0, 50.0, 60.0에 대한 Vc, RPM, Feed 값이 정리되어 있다. 저탄소강, 일반강, 고탄소강, 스테인리스강, 주철, 알루미늄 합금, 비철금속, 티타늄 합금까지 폭넓은 피삭재 조건을 포함한다.
