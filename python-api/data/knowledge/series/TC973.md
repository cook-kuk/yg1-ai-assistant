# YG-1 — TC973
**제품명**: SCREW THREAD INSERT TAP

## 기본 사양
- 공구 타입: Tap
- 서브타입: Screw Thread Insert Tap
- 날 수: 2
- 공구 재질: HSS-E
- 코팅: Bright
- 생크: DIN 371 / DIN 376
- 직경 범위: M2.5~M20
- 깊이비: 3.0×D

## 대응 ISO 그룹
P, N

## 대상 피삭재
- Non-alloy steel
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)
- AI

## 특징
- Recommended cutting conditions are provided for this tap series.
- Machine taps
- Wire insert threads are used for increasing fastening strength in soft materials.
- DIN 371 (M2.5~M8) and DIN 376 (M10~M20)
- Hole type: 3.0×D
- 6H Mod.
- 60° thread form

## 용도
- Thread tapping
- Tapping for screw thread inserts / helicoil-type threads
- Machine tapping in soft materials

## 상세 정보 (원문 요약)
- [page_0943] SCREW THREAD INSERT TAP의 추천 절삭조건 페이지입니다. TC909/TC944와 TC973/TC934/TC954에 대해 ISO P, N 재질별 Vc(m/min.)가 제시되어 있으며, 비합금강, 알루미늄(단조/주조), 구리 및 구리합금에 대한 절삭속도 범위를 확인할 수 있습니다.
- [page_0406] SCREW THREAD INSERT TAP TC973 SERIES는 EG-M ISO metric coarse threads for Screw Thread insert용 머신 탭이다. 와이어 인서트 나사는 연질 재질에서 체결 강도를 높이기 위해 사용되며, Hole type은 3.0×D이다. DIN 371(M2.5~M8)과 DIN 376(M10~M20) 규격으로 제공되며 Bright 마감이다.
