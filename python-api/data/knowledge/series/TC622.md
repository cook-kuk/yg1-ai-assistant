# YG TAP ALU — TC622
**제품명**: TC622 SERIES M-Az Machine taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 날 수: 3
- 직경 범위: M2~M22
- 깊이비: 3.0×D

## 대응 ISO 그룹
P, N

## 대상 피삭재
- Non-alloy steel
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- ISO metric coarse threads DIN 13
- Interrupted tap to reduce contact area and tapping torque
- Provides more chip space
- DIN 371 / DIN 376
- Hole type: 3.0×D
- * DIN profile not ISO

## 용도
- General machine tapping

## 상세 정보 (원문 요약)
- [page_0363] TC622 SERIES는 M-Az ISO metric coarse thread용 머신 탭입니다. 절삭날이 끊어진(interrupted) 형상으로 접촉면과 탭핑 토크를 줄이고 칩 공간을 늘리도록 설계되었습니다. DIN 371 / DIN 376 규격과 Hole type 3.0×D가 제시되며, 표에는 M2부터 M22까지의 규격과 피치, EDP No., 치수, 플루트 수, 드릴 직경이 포함됩니다.
- [page_0366] 추천 절삭조건 페이지로, YG TAP ALU의 재질별 권장 절삭조건을 제시한다. 상단 헤더에 TC163, TC963, TC169, TC170, TE953, TC622, TE943, TC433, TE443, TY433 시리즈가 표시되어 있으며, 본 페이지는 숫자 절삭조건 표 중심이다. 제품 상세 설명은 없고, ISO P/N 재질군에 대한 적용 재질만 확인된다.
