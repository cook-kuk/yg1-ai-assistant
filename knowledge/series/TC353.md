#  — TC353
**제품명**: TC353 SERIES / ISO metric coarse threads DIN 13 / Sets of taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Hand tap set
- 날 수: 3, 4
- 직경 범위: M3~M20
- 깊이비: 2.0xD

## 대응 ISO 그룹
P, M, K, N

## 대상 피삭재
- Non-alloy steel
- Low alloy steel
- Stainless steel
- Grey cast iron
- Nodular cast iron
- Aluminum-wrought alloy
- Aluminum-cast, alloyed
- Copper and Copper Alloys (Bronze / Brass)

## 특징
- Sets of taps in First, Second and Bottoming
- Bottoming tap of set has final internal thread dimensions only
- First with pilot guide
- Hole type 2.0xD

## 용도
- ISO metric coarse threads DIN 13
- Hand tapping in set

## 상세 정보 (원문 요약)
- [page_0397] TC353 SERIES는 ISO metric coarse threads DIN 13용 탭 세트이며, First, Second, Bottoming으로 구성된 수동 핸드 탭입니다. Bottoming tap은 최종 내나사 치수만 가공하며, First는 pilot guide가 있습니다. Hole type은 2.0xD이고, 규격은 M3부터 M20까지 제공됩니다.
