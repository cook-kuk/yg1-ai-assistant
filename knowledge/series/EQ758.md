# YG-1 — EQ758
**제품명**: 표준 / 짧은길이 / 롱 / 엑스트라 롱 라핑 (TiAlN)

## 기본 사양
- 공구 타입: End Mill
- 서브타입: Roughing
- 공구 재질: M42 HSS
- 코팅: TiAlN
- 직경 범위: 6.0~50.0

## 대응 ISO 그룹
P, N

## 대상 피삭재
- 탄소강
- 합금강
- 공구강
- 알루미늄
- 알루미늄 합금

## 특징
- 표준 / 짧은길이 / 롱 / 엑스트라 롱 라핑용 시리즈

## 용도
- 라핑 가공

## 상세 정보 (원문 요약)
- [page_0658] 추천 절삭조건 M42 HSS 페이지로, 표준 / 짧은길이 / 롱 / 엑스트라 롱 라핑 (TiAlN)용 EQ758 / EQ760 / EQ759 / EQ751 / EQ714 / EQ752 시리즈의 절삭조건을 제시한다. 본문은 ISO P와 N 재질에 대한 Vc, Fz, RPM, Feed, Ap, Ae 수치표이며, Ap는 1.5D, Ae는 0.1D 조건이 표시되어 있다. 알루미늄 및 알루미늄 합금도 포함되어 있다.
