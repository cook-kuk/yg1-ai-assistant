# YG TAP STEEL — TY422
**제품명**: TY422 SERIES / ISO metric coarse threads DIN 13 / Machine taps

## 기본 사양
- 공구 타입: Tap
- 서브타입: Machine tap
- 날 수: 3-4
- 코팅: TiAlN
- 생크: DIN 371 / DIN 376
- 직경 범위: M2~M30
- 깊이비: 3.0×D

## 특징
- Suitable for through hole
- Higher cutting speed than other taps due to thick web
- DIN 371 for M2~M10 and DIN 376 for M11~M30
- DIN profile not ISO
- Hole type: 3.0×D

## 용도
- ISO metric coarse thread tapping
- Machine tapping
- Through-hole tapping

## 상세 정보 (원문 요약)
- [page_0312] TY422 시리즈는 ISO metric coarse threads DIN 13용 머신 탭이며, 두꺼운 웹(thick web) 구조로 인해 다른 탭보다 높은 절삭속도에서 사용하기에 적합하고 관통홀(through hole)에 적합하다고 설명되어 있습니다. DIN 371(M2~M10)과 DIN 376(M11~M30) 규격을 따르며, Hole type은 3.0×D입니다. 표에는 M2부터 M30까지의 규격과 EDP No., 치수 정보가 수록되어 있습니다.
