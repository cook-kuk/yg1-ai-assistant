# YG-1 — D1201 / D1206 / D1203 / D1202 / D1204 / D1208 / D1207 / D1209 / D1210 / D1205 / D1953
**제품명**: 하이스 드릴 / 테이퍼 생크 드릴, 노스 드릴

## 기본 사양
- 공구 타입: Drill
- 서브타입: Taper Shank Drill / Morse Taper Drill
- 공구 재질: HSS
- 생크: Taper shank
- 직경 범위: D13.0~D20.0

## 대응 ISO 그룹
P, M, K, N, S

## 대상 피삭재
- P
- M
- K
- N
- S

## 특징
- 추천 절삭조건 페이지
- ● : BEST, ○ : GOOD
- 드릴 직경 13.0, 16.0, 18.0, 20.0 기준 조건 제시

## 용도
- 저탄소강, 일반강, 중탄소강, 고탄소강, 저합금강, 고합금강 및 고합금 공구강
- 스테인레스 강
- 회주철, 구상흑연주철, 가단 주철
- 알루미늄-단조 합금, 알루미늄-주조 합금, 비철금속
- 티타늄 합금

## 상세 정보 (원문 요약)
- [page_1003] 추천 절삭조건 페이지로, 하이스 드릴의 테이퍼 생크 드릴/노스 드릴 D1201 / D1206 / D1203 / D1202 / D1204 / D1208 / D1207 / D1209 / D1210 / D1205 / D1953 시리즈에 대한 절삭조건을 제시한다. ISO P, M, K, N, S 재질군별로 Vc, RPM, Feed를 드릴 직경 13.0, 16.0, 18.0, 20.0 기준으로 안내하며, ●는 BEST, ○는 GOOD을 의미한다. 저탄소강부터 티타늄 합금까지 다양한 피삭재에 대한 추천 조건이 포함되어 있다.
